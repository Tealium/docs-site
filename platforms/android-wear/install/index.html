ANDROID WEAR

Install

Get the code and install the Tealium for Android Wear SDK.

Requirements

  • Tealium Universal Data Hub account
  • Android app with Tealium for Android
  • Android Version 4.4, API Level 20 or higher (special version for Wear)

Sample Apps

We recommend downloading the AndroidWear sample app we provide within the main Android repository. This will help to familiarize yourself with our library, the tracking methods, and best practice implementation.

  • AndroidWear Sample App

We also recommend the use of a helper class to abstract the Tealium implementation from the rest of your app. This will give you a single entry point from which to initialize and make tracking calls. This approach allows you to update the code in your helper file and not in every single Java file.

  • Sample Helper Class

Install

Download and install the library before proceeding.

  • Download: Tealium for Android Wear

Handheld

Add tealium.mobile-5.x.x.aar to your application’s dependencies. No additional code is necessary.

Wearable

In the application’s wear module add the tealium.wear-5.x.x.aar dependency. Subclass the Wear Application’s subclass and add the TealumWear’s initialization code in its onCreate function.

public class WearApp extends Application {

    public static final String TEALIUM_MAIN = "INSTANCE"; // Instance name from main app

    @Override
        public void onCreate() {
            super.onCreate();

        TealiumWear.createInstance(
            TEALIUM_MAIN,
            TealiumWear.Config.create(this)
                .setLogLevel(Log.VERBOSE));
    }
}

The value of TEALIUM_MAIN must match the instance name used in the initialization of the main application. Example:

// Main application initialization
Tealium.createInstance("my_app", config);

// WearApp initialization
    public static final String TEALIUM_MAIN = "my_app";

Track Views

Tracking on the wearable is similar to the handheld side. The only difference is in how to pass custom data to the tracking calls. The wearable side uses a DataMap() instead of a Map():

java
final DataMap data = new DataMap();
// ...

TealiumWear.getInstance(WearApp.TEALIUM_MAIN)
    .trackView("someView", data);

Track Events

java
final DataMap data = new DataMap();
// ...

TealiumWear.getInstance(WearApp.TEALIUM_MAIN)
    .trackEvent("someEvent", data);
