## API for Android/iOS

The following resources provide links to Tealium API reference for Android and iOS. If you are building a mobile application in Xamarin, use bindings to our iOS and Android libraries.

*   [Tealium for iOS](https://docs.tealium.com/platforms/ios-objective-c/api/)
*   [Tealium for Android](https://docs.tealium.com/platforms/android-java/api)

## API for C\#

Alternatively, if only using Tealium Customer Data Hub services (Tealium AudienceStream, Tealium EventStream, DataAccess), the [Tealium for C#](https://docs.tealium.com/platforms/csharp/api) library may be used as the native code in a Xamarin project.
