Tealium for iOS (Swift) version 1.x

Swift library for the Tealium Collect service.

The latest Tealium for iOS is Swift 2.x. Swift 1.x and Objective-C are the previous versions. Swift code co-exists along side your existing Objective-C files in the same project, with full access to your Objective-C API, making it easy to adopt.
