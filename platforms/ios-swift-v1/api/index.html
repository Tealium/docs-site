API Reference

Reference guide for classes and methods provided by Tealium for iOS (Swift).

Tealium Swift Classes

TealiumConfig

A class to set configuration options for the main Tealium class. Individual modules provide their own extensions for the TealiumConfig class if they are enabled.

TealiumConsentCategories

Provides methods for Tealium consent categories.

TealiumConsentManagerDelegate

Provides methods for the Tealium Consent Management delegate.

TealiumConsentManager

Provides methods for Tealium Consent Management.

TealiumInstanceManager

Provides a method of accessing a previously-instantiated instance of Tealium, using the key format “account.profile.environment”.

TealiumPersistentData

Provides methods for accessing persistent data.

TealiumVolatileData

Provides methods for accessing volatile data.

Tealium

The Tealium class serves as the main API entry point for all modules.
