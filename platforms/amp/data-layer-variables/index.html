AMP

Data Layer

Full list of data layer variables available from the AMP library.

The following parameters are populated by the Tealium AMPÂ Library. These variables are included automatically in each tracking call from the library and can be used in your Universal Data Hub configuration.

In the following table, some variables are used for both page view/event tracking, while some are only used for each.

For page view tracking, the default event specification for tealium_event is "screen_view". For event tracking, each trigger should specify an event name for tealium_event.

Parameter Description Page Tracking Event Tracking
amp_hostname AMP hostname
Example: example.com		 ✔ ✔
amp_request_count AMP request count
Example: 1		 ✔ ✔
amp_version AMP version number
Example: 1907022322580		 ✔ ✔
ampdoc_url AMP document URL
Example: http://example.com/path/to/file.html		 ✔ ✔
canonical_hostname Canonical hostname
Example: example.ampproject.org		 ✔ ✔
content_load_ms Time in milliseconds to load the content
Example: 66		 ✔
domain The full domain name of the URL
Example: example.com		 ✔ ✔
lang Language code
Example: en-us		 ✔ ✔
page_view_id Page view ID number
Example: 769		 ✔
pathname Path of the URL, excluding the query parameters and domain
Example: /path/to/file.html		 ✔ ✔
referrer URL path without the file name
Example: http://example.com/path/to/		 ✔
screen_size Screen resolution height x width (pixels)
Example: 2560x1440		 ✔
scroll_x Horizontal scrolling value (x-axis)
Example: 0		 ✔
scroll_y Vertical scrolling value (y-axis)
Example: 0		 ✔
tealium_visitor_id Generated uuid (case-sensitive)
Example: amp-ABc1dEF2gH3IJK4lmn5OPz		 ✔ ✔
timestamp Epoch timestamp (can be overridden in the vars block)
Example: 1563305241134		 ✔ ✔
title AMP document’s title (can be overridden in the vars block)
Example: Hello, AMP		 ✔ ✔
tz Time zone
Example: 420		 ✔ ✔
url The full URL of the page
Example: http://example.com/path/to/file.html		 ✔ ✔
viewport_height Browser viewport height (pixels)
Example: 1417		 ✔ ✔
viewport_width Browser viewport width (pixels)
Example: 2560		 ✔ ✔
