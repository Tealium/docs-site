## Tag configuration

Go to the tag marketplace to add a new tag. For more information about how to add a tag, see [Manage tags](https://docs.tealium.com/manage-tags/).

When adding the tag, configure the following settings:

* **Account**: (Required) The Visual Website Optimizer account ID.

## Load rules

Load the tag on all pages or set conditions for when the tag loads. For more information, see [About load rules](https://docs.tealium.com/about-load-rules/).

## Data mappings

No data mappings are available for this tag.
