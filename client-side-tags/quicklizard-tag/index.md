## Tag Tips

* Required mappings:
  * List of Product IDs
  * List of Prices

* Supports the following E-Commerce extension values:
  * Order ID
  * List of Product IDs
  * List of Names
  * List of Brands
  * List of Categories
  * List of Prices

## Tag Configuration

First, go to Tealium&#39;s tag marketplace and add the Quicklizard tag (Learn more about [how to add a tag]()).

After adding the tag, configure the following settings:

* **Client Key**  
The key supplied by Quicklizard.
