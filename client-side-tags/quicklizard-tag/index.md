## Tag Tips

* Required mappings:
  * List of Product IDs
  * List of Prices

* Supports the following E-Commerce extension values:
  * Order ID
  * List of Product IDs
  * List of Names
  * List of Brands
  * List of Categories
  * List of Prices

## Tag Configuration

First, go to Tealium's tag marketplace and add the Quicklizard tag (Learn more about [how to add a tag](https://docs.tealium.com/manage-tags/)).

After adding the tag, configure the following settings:

* **Client Key**  
The key supplied by Quicklizard.

## Data Mappings

Mapping is the process of sending data from a [data layer variable](https://docs.tealium.com/data-layer-variables/) to the corresponding destination variable of the vendor tag. For instructions on how to map a variable to a tag destination, see [Data Mappings](https://docs.tealium.com/iq-tag-management/data-mappings/manage/).
