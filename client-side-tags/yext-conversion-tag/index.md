---## Tag tips

* Use mapping to override the standard configuration values.
* Conversions will fire if the ID mapping is populated.

## Tag configuration

Go to the tag marketplace to add a new tag. For more information about how to add a tag, see [Manage tags](https://docs.tealium.com/manage-tags/).

## Data mappings

Mapping is the process of sending data from a [data layer variable](https://docs.tealium.com/data-layer-variables/) to the corresponding destination variable of the vendor tag. For instructions on how to map a variable to a tag destination, see [Data Mappings](https://docs.tealium.com/iq-tag-management/data-mappings/manage/).

The available categories are:

### Standard

|Variable| Description|
|---| ---|
|`optin`|  <ul><li>Optin</li></ul> |
|`cid`|  <ul><li>Customer Identification Number (CID)</li></ul> |
|`value`|  <ul><li>Value</li></ul> |
