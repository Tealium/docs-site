To use the Adobe Test & Target tool in Web Companion:

1. Click on the Web Companion bookmarklet to open Web Companion.
1. Navigate to the Tools tab.
    ![](https://docs.tealium.com/images/iq-tag-management/no-title-1319if034cb2cd6b3f073.png)

1. Click on the Adobe Test and Target bar.
1. Click **Select Target**. The On Page Element Selector crosshairs are displayed.
1. With those crosshairs, select the element you want to apply a test to.
1. Web Companion will populate the identifying information for the element you selected. Enter the mBox info and Static params.
    ![](https://docs.tealium.com/images/iq-tag-management/no-title-1318i318bf7765a3d7dfb.png)

1. Select the placement for the modifying content. For more information on this see the [Test &amp; Target](https://docs.tealium.com/adobe-test-target-tag/) article.
1. Check the box next to Flicker Free to use Tealium's Flicker Free solution. See the [Flicker Free Test and Target](https://docs.tealium.com/flicker-free-test-and-target/) article.
1. From the drop-down at the bottom of Web Companion, select the Tag you want to scope this Extension to.
1. Click **New Tag** to add a new instance of the Test and Target Tag.
1. Click **Add to Queue**.
1. Navigate to the Queue and click **Save**.
1. Enter your Tealium iQ login information and click **Login**.
1. By default, anything added with Web Companion is turned off in Tealium iQ. Log in to Tealium iQ and turn on the tag and/or extension you added.
1. Save/Publish the profile.
