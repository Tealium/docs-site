## What&#39;s new

This is an update to Tealium Tools browser extension available on the Google Chrome Web Store and as a Microsoft Edge Add-on.

To install Tealium Tools Version 3, see .

* **Tool Catalog**:  
The **Tool Catalog** replaces Custom tools and now includes community tools grouped under this tab.

* **Support for Manifest V3**:  
Google Chrome is rolling out an important update that affects all Chrome extensions. This update removes the extensions&#39; ability to load custom JavaScript from external sources, such as Tealium’s custom tool repository server. This enforced update aims to provide Chrome users with additional security by preventing extensions from loading potentially malicious content that isn&#39;t contained within the extension itself. For more information, see [Chrome for Developers: Extensions &gt; Manifest V3](https://developer.chrome.com/docs/extensions/develop/migrate/what-is-mv3).

* **Sidebar support**:  
You can now open the tools window in the Chrome sidebar, allowing you to work seamlessly alongside on-page content. This is particularly useful for tools like **Web Companion**, which can now run unobstructed while you interact with your page.

* **Custom tile sorting**:  
You can now sort and retain the tile order in both tabs. For accessibility, we’ve implemented a &#34;Click-and-Hold to Unlock&#34; feature similar to iOS. Hold the left mouse button on a tile for 3 seconds to enable &#34;Wobble Mode&#34; and rearrange tiles. This prevents accidental reordering while improving usability.

* **SSO login support**:  
Single Sign-On (SSO) functionality is now more reliable. You can set your SSO domain and log in seamlessly – just ensure SSO is enabled on your account. Password and MFA logins are still supported. For SSO enablement, contact your Customer Success Manager.

* **uTag Debugger**:  
We’re excited to include the **uTag Debugger** as an official tool on the **Home** tab. It’s a valuable resource for troubleshooting and validating deployments.

* **Built-in feedback form**:  
In the admin section (accessed by clicking your user avatar), you’ll find two new tabs: **About** and **Feedback**. The Feedback tab currently supports bug submissions, pre-populating tool information to help us troubleshoot effectively. The **About** tab provides build details for support tickets and information about community tools.

* **Retired features**:  
The following features have been removed:
  * **Multi-Publish**
  * **CDH Copy/Paste**
  * **Scan Companion**
  * **Verify - Test Recorder**
  Please contact support if your workflow has been affected by this change.

* **Updated tools**:  
  * **Tealium University**, **Tealium Docs**, and **UTag Debugger** are now available as separate launch tiles.
  * The **TLC** tile has been renamed to **Knowledge Base**.

* **Early Access tools**:  
Early Access tools now include a callout bubble, indicating they are publicly available but still in the early stages of release.
