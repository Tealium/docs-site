## How it works

Your account is organized into profiles. A profile corresponds to a website, a set of websites with similar or identical tagging requirements, or a mobile application. The names of your account and profiles are referenced in the installation code for the websites and the mobile applications.

 Only client-side profiles can be created through the platform. To create a server-side profile, contact Tealium Support. 

### Profile libraries

A library is identical to a profile, but with the special ability to be imported into another profile. Libraries are used to manage configurations that are shared across multiple profiles.

For additional information, see [Managing Profile Libraries](/iq-tag-management/profiles/profile-libraries/manage/).
