---
Select **Resource Center** in the [Tealium platform](https://my.tealiumiq.com/) header to open the Resource Center panel. The panel gives you access to help, training, support, and feedback resources without leaving the platform.

![](/images/help/resource-center.png)

| Module | Description |
|---|---|
| Product Guides | Shows documentation relevant to your current location in the platform. Documentation updates as you navigate to different screens. Use the search box to search across all product documentation. |
| Knowledge Base | Links to the [Tealium support knowledge base](https://support.tealiumiq.com/en/support/solutions), which covers user guides, troubleshooting, implementation strategies, and FAQs. |
| Feedback | Browse, vote on, and submit product ideas in the Tealium Ideas Portal. For more information, see [Tealium feedback policy](). |
| Product Updates | Displays in-product announcements about new Tealium releases and feature updates. |
| Case Studies &amp; Events | Features in-product content about upcoming webinars, Tealium events, and customer success stories. |
| Training &amp; Certification | Links to [Tealium University](https://university.tealium.com/learn), where you can access on-demand courses and certification programs. |
| Support Desk | Links to the [Tealium support portal](https://support.tealiumiq.com/en/support/tickets), where you can submit a support ticket. |
