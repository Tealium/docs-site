## EventStream API Hub


![](/images/icons/icon-product-eventstream.svg)

#### 仕組み

オンラインおよびオフラインのデータ収集を管理し、データ品質を保証し、サーバーサイドの分析およびマーケティングツールとの統合を行います。



![](/images/icons/icon-arrow-right-into.svg)

#### データソース

ウェブサイト、モバイルアプリ、カスタムアプリケーションからTealiumにデータを送信します。



![](/images/icons/icon-product-data-connect.svg)

#### データコネクト

内部システムからのデータをTealiumに統合して、イベントと訪問のエンリッチメントを行います。



![](/images/icons/icon-file-arrow-up.svg)

#### ファイルインポート

オフラインおよびオンラインチャネルから顧客データをインポートします。



![](/images/icons/icon-cubes.svg)

#### 属性

イベントと訪問の基本データ要素を管理します。



![](/images/icons/icon-cubes-check.svg)

#### 仕様

受信データの品質を整理し、検証します。



![](/images/icons/icon-arrows-merge.svg)

#### フィード

最も重要なイベントタイプのフィードを定義します。


## AudienceStream CDP


![](/images/icons/icon-product-audiencestream.svg)

#### 仕組み

顧客プロファイルを統合し、カスタムオーディエンスを構築し、リアルタイムのパーソナライゼーションを管理します。



![](/images/icons/icon-moments-api.svg)

#### Moments API

高性能データをリアルタイムで訪問のパーソナライゼーションに活用するためのコンポーザブルAPIを使用します。



![](/images/icons/icon-cubes.svg)

#### 属性

イベントと訪問の基本データ要素を管理します。



![](/images/icons/icon-audience.svg)

#### オーディエンス

最も価値のある顧客を特定するためのオーディエンスセグメントを作成します。



![](/images/icons/icon-id.svg)

#### アイデンティティ

訪問のスティッチングを使用して顧客の統合ビューを作成します。


## CloudStream


![](/images/icons/icon-cloud-down.svg)

#### 仕組み

データクラウドから直接顧客データを活用し、Tealiumにデータを保存またはロードすることなく操作します。




![](/images/icons/icon-audience.svg)

#### セグメント

データクラウドからのレコードセットを定義し、アクションを実行するためのセグメントを作成します。


## インテグレーション


![](/images/icons/icon-connector.svg)

#### コネクタ

コネクタを使用して、イベントおよび訪問データをお好みのベンダーAPIと統合します。



![](/images/icons/icon-code.svg)

#### Webhooks

Webhookコネクタを使用してカスタムインテグレーションを構築します。

## その他の機能


![](/images/icons/icon-trace.svg)

#### トレース

構成をテストし、イベントや顧客データがリアルタイムでどのように処理されるかを確認します。



![](/images/icons/icon-bolt.svg)

#### 関数

受信データを変更するための独自のJavaScript関数を記述するか、リアルタイムでイベントやオーディエンスを処理します。



![](/images/icons/icon-product-dataaccess.svg)

#### Tealium DataAccess

Tealium DataAccessを使用して、長期データ保存をソリューションに拡張します。



![](/images/icons/icon-curly-brackets.svg)

#### EventStoreとAudienceStore

イベントデータと訪問データをJSONファイルとして直接アクセスします。



![](/images/icons/icon-database.svg)

#### EventDBとAudienceDB

イベントデータと訪問データを構造化データベースで直接アクセスします。



![](/images/icons/prod_icon_din.svg)

#### Tealium Insights

EventDBとAudienceDBからのデータを表示するダッシュボードを閲覧します。



![](/images/icons/icon-gear.svg)

#### ユーザー権限

アカウントとユーザー権限を管理します。
