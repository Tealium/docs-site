## SFTP

### 資格情報

ファイルインポート構成でファイルサービスを構成するためには、以下のFTP資格情報が必要です：

* ホスト名
* ユーザー名
* 認証方法  
**パスワード**、**プライベートキーファイルのアップロード**、および**キーペアの生成**がサポートされている認証方法です。

さらに、ポートを指定し、サブディレクトリを追加することができます。

## FTP

FTP接続を使用してファイルインポートデータソースに接続すると、FTP資格情報を含むすべてのデータがインターネット上で暗号化されずに送信されます。

### 資格情報

ファイルインポート構成でファイルサービスを構成するためには、以下のFTP資格情報が必要です：

* ホスト名
* ユーザー名
* パスワード
* ポート

さらに、サブディレクトリを追加することができます。
