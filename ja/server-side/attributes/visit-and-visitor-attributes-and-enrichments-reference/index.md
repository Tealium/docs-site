## バッジ

* 訪問レベルの属性
* Webhookで使用できます
* データレイヤーエンリッチメントキー名：`badges`
* 詳細については、[バッジ属性]()を参照してください。

### エンリッチメント

|エンリッチメント| ソース値|
|---| ---|
|割り当て|
|削除|
|別のバッジから割り当て| |

### ルール条件

|ルール条件| ソース値|
|---| ---|
|割り当てられている&lt;br&gt; 割り当てられていない|

## 文字列のセット

* 訪問および訪問レベルの属性
* Webhookで使用できます
* データレイヤーエンリッチメントキー名：`property_sets`
* 詳細については、[文字列のセット属性]()を参照してください。

### エンリッチメント

|エンリッチメント| ソース値|

|追加| |
|配列を文字列のセットとして保存| |
|別の文字列のセットによってこの文字列のセットを更新| |
|文字列のセットAに存在し、文字列のセットBに存在しない値を構成| |
|すべてのエントリを削除|
|1つのエントリを削除| |
|すべてのエントリを小文字に|
| 値に基づいて集計から上位Nキーを構成 | |
| 値がXより大きい集計からキーを構成 | |

### ルール条件

|ルール条件| ソース値|
|---| ---|
|割り当てられている&lt;br&gt; 割り当てられていない| |
| 含む &lt;br&gt; 含む (キー) | |
| 部分文字列を含む&lt;br&gt; 部分文字列を含む（大文字小文字を無視）&lt;br&gt; 部分文字列を含む（キーに部分文字列が含まれる）&lt;br&gt; 部分文字列を含む（大文字小文字を無視）（キーに部分文字列が含まれる） | |
| 含まない&lt;br&gt; 含まない（キー）&lt;br&gt; 含まない（キー）（大文字小文字を無視）&lt;br&gt; 含まない（大文字小文字を無視） | |

## Tally

* 訪問レベルと訪問レベルの属性
* Webhookで使用可能
* データレイヤーのエンリッチメントキー名：`metric_sets`
* 詳細は、[Tally Attribute]()を参照してください。

### エンリッチメント

|エンリッチメント| ソース値|
|---| ---|
|キーの増分| |
|値によるキーの増分| |
|他のtallyのすべてのキーと値による増分| |
| 配列に基づいたtallyの構成（一つの配列にキー、別の配列に値） | |
| tallyの削除 |
| tallyからキー値ペアの削除（キー） |    |
| 各項目についてtallyの値を1増分 | |
| tallyをキャプチャしたtallyのローリング合計/平均に構成 | |

### ルール条件

|ルール条件| ソース値|
|---| ---|
|割り当てられている&lt;br&gt; 割り当てられていない|
|キーを含む（キーの値をチェックするオプション付き）| |
|部分文字列を含む（キーがこの部分文字列を含む）。大文字小文字を無視するオプションと、キーの値をチェックするオプション付き。| |
|含まない&lt;br&gt; 含まない（キー）&lt;br&gt; 含まない（キー）（大文字小文字を無視）&lt;br&gt; 含まない（大文字小文字を無視）| |

## 数値

* 訪問-および訪問レベルの属性
* ウェブフックで使用可能
* データレイヤーのエンリッチメントキー名：`metrics`
* 数値は小数または整数であることができます。整数は最も近い整数に切り上げまたは切り捨てられます。（実験モード）
* 詳細は、[数値属性]()を参照してください。

### エンリッチメント

|エンリッチメント| ソース値|
|---| ---|
|増加/減少| |
|比率/積/差/合計| |
|数値を構成| |
|（他の）数値の変化| |
|項目の数| |
|特定の（ハードコーディングされたキー）値| |
|項目の数| |
|格納された数値の移動平均/合計/最大値| |
|エントリの数| |
|2つの日付間の差 (`mhdwm`)| |
|日付の変更 (`mhdwm`)| |
|配列内の項目数に構成| |

### ルール条件

|ルール条件| ソース値|
|---| ---|
| 等しい&lt;br&gt; 等しくない | |
|割り当てられている&lt;br&gt; 割り当てられていない|
|以上&lt;br&gt; 以下| |
|変更された|
