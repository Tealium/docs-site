title: 人気のバッジ
description: これらは他のクライアントに役立つことが証明された人気のバッジです。

---
| バッジ | 説明 |
|:-----:| ----------- |
| ![](/images/server-side/whiteui-badge-browseabandoner.png)&lt;br&gt; ブラウズ放棄者 |  &lt;ul&gt;&lt;li&gt;**ルール** – 商品を閲覧し、カートに商品を追加しなかった訪問。&lt;/li&gt;&lt;li&gt;**アクション** – 既知の訪問であり、Facebook IDが分かっている場合、データをFacebookに送信して、ブランドを訪問に再ターゲティングします。&lt;/li&gt;&lt;/ul&gt; |
| ![](/images/server-side/whiteui-badges)business-family.png)&lt;br&gt; ビジネス/ホーム族旅行者 |  &lt;ul&gt;&lt;li&gt;**ルール** – 訪問が旅行のタイプをマークしたときに割り当てます。&lt;/li&gt;&lt;/ul&gt; |
| ![](/images/server-side/whiteui-badges-cartabandoner.png)&lt;br&gt; カート放棄者 |  &lt;ul&gt;&lt;li&gt;**ルール** – カートに商品を追加し、購入しなかった訪問。&lt;/li&gt;&lt;li&gt;**アクション** – メールアドレスが分かっている場合、放棄された商品を含むパーソナライズされたメールを送信します。&lt;/li&gt;&lt;/ul&gt; |
| ![](/images/server-side/whiteui-badges-couponcodeabuser.png)&lt;br&gt; クーポンコード乱用者 |  &lt;ul&gt;&lt;li&gt;**ルール** – 訪問が一度の訪問で特定の数以上のクーポンまたはプロモーションコードを使用しようとする。&lt;/li&gt;&lt;/ul&gt; |
| ![](/images/server-side/whiteui-bades-couponcodestuffer.png)&lt;br&gt; クーポンコード詰め込み者 |  &lt;ul&gt;&lt;li&gt;**ルール** – 訪問がすべての訪問で特定の数以上のクーポンまたはプロモーションコードを成功裏に使用している。&lt;/li&gt;&lt;/ul&gt; |
| ![](/images/server-side/whiteui-badges-disappointedshopper.png)&lt;br&gt; がっかりしたショッパー |  &lt;ul&gt;&lt;li&gt;**ルール** – 現在在庫切れの商品を閲覧した訪問に割り当てます。&lt;/li&gt;&lt;/ul&gt; |
| ![](/images/server-side/whiteui-badges-freetrialuser.png)&lt;br&gt; 無料トライアルユーザー |  &lt;ul&gt;&lt;li&gt;**ルール** – 無料トライアルにサインアップした訪問に割り当てます。&lt;/li&gt;&lt;/ul&gt; |
| ![](/images/server-side/whiteui-badges-frequenttraveler.png)&lt;br&gt; 頻繁な旅行者 |  &lt;ul&gt;&lt;li&gt;**ルール** – 訪問が特定の期間に特定の回数以上旅行したときに割り当てます。例：訪問が過去12週間で5回旅行した。&lt;/li&gt;&lt;/ul&gt; |
| ![](/images/server-side/whiteui-badges-knownvisitor.png)&lt;br&gt; 既知の訪問 |  &lt;ul&gt;&lt;li&gt;**ルール** – 訪問の重要な識別子をキャプチャしたときに割り当てます。例：顧客ID（クライアントのID）、顧客のメール、Facebook ID。これらの訪問は通常、メールやソーシャルメディアを通じてターゲットにされます。&lt;/li&gt;&lt;li&gt;**アクション** – なし。ただし、選択したベンダーAPIで訪問がターゲットにできることを確認するために、オーディエンスで使用されます。&lt;/li&gt;&lt;/ul&gt; |
| ![](/images/server-side/whiteui-badges-loyalvisitor.png)&lt;br&gt; 忠実な訪問 |  &lt;ul&gt;&lt;li&gt;**ルール** – サイトへの直接訪問が特定の回数に達した訪問に割り当てます。&lt;/li&gt;&lt;/ul&gt; |
| ![](/images/server-side/whiteui-badges-newvisitor.png)&lt;br&gt; 新規訪問 |  &lt;ul&gt;&lt;li&gt;**ルール** – &#34;ライフタイム訪問数&#34;が1に等しいときに割り当てます。&lt;/li&gt;&lt;/ul&gt; |
| ![](/images/server-side/whiteui-badges-purchaser.png)&lt;br&gt; 購入者 |  &lt;ul&gt;&lt;li&gt;**ルール** – コンバートした訪問。例：商品を購入した、またはサービスにサインアップした。&lt;/li&gt;&lt;li&gt;**アクション** – 2週間後にメールで別の購入を誘うオファー、または購入した商品に関連する商品を示すメールを送信します。例えば、訪問が野球のバットを買った場合、野球のバッグとバッティンググローブを提供します。&lt;/li&gt;&lt;li&gt;**アクション** – 訪問をリターゲティングキャンペーンから削除し、不必要に再マーケティングされないようにします。&lt;/li&gt;&lt;/ul&gt; |
| ![](/images/server-side/whiteui-badges-recentvipvisitor.png)&lt;br&gt; 最近のVIP訪問 |  &lt;ul&gt;&lt;li&gt;**ルール** – その90日間の購入価値が特定のドル額を超えたときに割り当てます。&lt;/li&gt;&lt;/ul&gt; |
| ![](/images/server-side/whiteui-badges-salesearcher.png)&lt;br&gt; セール検索者 |  &lt;ul&gt;&lt;li&gt;**ルール** – 訪問が特定の数以上のセールページを閲覧したときに割り当てます。それがセールスカテゴリーであるか、セール中の商品であるかは問いません。&lt;/li&gt;&lt;/ul&gt; |
| ![](/images/server-side/whiteui-badge-browseabandoner.png)&lt;br&gt; 検索放棄者 |  &lt;ul&gt;&lt;li&gt;**ルール** – 検索エンジンからサイトにアクセスし、共通のアクションを完了していない訪問。例えば、訪問中に少なくとも3ページを閲覧するなど。&lt;/li&gt;&lt;li&gt;**アクション** – 検索したカテゴリーを特定し、ディスプレイ広告ベンダーで再ターゲティングします。&lt;/li&gt;&lt;/ul&gt; |
| ![](/images/server-side/whiteui-badges-siteresearcher.png)&lt;br&gt; サイトリサーチャー |  &lt;ul&gt;&lt;li&gt;**ルール** – 訪問がサイトを特定の回数以上訪れ、平均訪問時間が特定の分数以上の場合に割り当てます。&lt;/li&gt;&lt;/ul&gt; |
| ![](/images/server-side/whiteui-badges-testuser.png)&lt;br&gt; テストユーザー |  &lt;ul&gt;&lt;li&gt;**ルール** – &#34;test&#34;がURLに含まれている、またはデバッグクッキーが構成されている、またはテスト訪問を示す他の基準がある場合に割り当てます。&lt;/li&gt;&lt;li&gt;キャンペーンがライブになる前に、構成が正しいことを確認するために、オーディエンスとアクションを製品でテストできます。&lt;/li&gt;&lt;/ul&gt; |
| ![](/images/server-side/whiteui-badges-unknownvisitor.png)&lt;br&gt; 未知の訪問 |  &lt;ul&gt;&lt;li&gt;**ルール** – 訪問がまだウェブサイトで認証していない場合に割り当てます。認証とは必ずしもログインを意味しません。訪問がメールからサイトに来て、URLにメールが含まれている場合、訪問が誰であるかを特定し、彼らが&#34;認証&#34;したと判断します。&lt;/li&gt;&lt;li&gt;**アクション** – 訪問に自分自身を特定するように促すモーダルを表示します。これらの訪問は、より一般的にディスプレイ広告を通じてターゲットにされます。&lt;/li&gt;&lt;/ul&gt; |
| ![](/images/server-side/whiteui-badges-recentvipvisitor.png)&lt;br&gt; VIP訪問 |  &lt;ul&gt;&lt;li&gt;**ルール** – ライフタイムの購入価値が特定のドル額に達したときに割り当てます。&lt;/li&gt;&lt;/ul&gt; |
| ![](/images/server-side/whiteui-badges-windowshopper.png)&lt;br&gt; ウィンドウショッパー |  &lt;ul&gt;&lt;li&gt;**ルール** – サイトを頻繁に訪れるが購入しない訪問。例：過去7日間に3回サイトを訪れ、商品詳細ページを閲覧したり記事を読んだりしているが、商品を購入したりサービスにサインアップしたりしていない訪問。&lt;/li&gt;&lt;li&gt;**アクション** – カテゴリーへの親和性を特定し、次のセッションのランディングページを変更して体験をパーソナライズします。&lt;/li&gt;&lt;/ul&gt; |
