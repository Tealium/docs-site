## 必要なもの

Tealium Predictを使用するためには以下の項目が必要です：

* **Tealium AudienceStream**  
Tealium Predict製品を使用するためには、データ収集が有効化されたTealium AudienceStream CDP製品が必要です。
* **データ量の最小限**  
過去のデータを使用して意味のあるモデルを構築するために十分なデータがあることを確認するため、以下のデータ量の最小限が必要です：
  * **21日間のデータ収集**  
AudienceStreamのデータ収集は、機能リリース前に最低21日間アクティブでなければなりません。
  * **最低平均日次訪問数**  
選択したトレーニング期間中に1日平均10,000回以上の訪問が必要です。
  * **ターゲット属性の最小ボリューム**  
ターゲット属性は、30日間にわたって真偽値がそれぞれ200以上の訪問者の日平均が必要です。
  * **データセットの期間**  
データセットのトレーニング期間：データセットのトレーニング期間は、予測期間プラス1日以上でなければなりません。例えば、「30日間での購入可能性」モデルでは、最低31日間のデータが必要です。より正確なトレーニングモデルを得るためには、UIで選択した予測期間に関係なく、トレーニング期間は90日間であるべきです。

## データ品質

最適な結果を得るためには、大量のトレーニングデータを含むクリーンなデータレイヤーで構成された高品質なデータセットがあることがTealiumから提案されています。より良いデータレイヤーの健康状態は、即時かつ意味のある予測結果を提供するモデルを作成することを可能にします。

詳細情報については、[機械学習のためのデータ準備](https://tealium.com/resource/fundamentals/data-readiness-for-ml-checklist/)を参照してください。

## プロフェッショナルサービス

製品の使用にはコーディングやデータサイエンスの知識は必要ありませんが、AudienceStreamの高品質な実装により結果を改善することができます。

データレイヤーの現状評価の支援が必要な場合、Tealiumは現在のAudienceStream実装と全体的なデータウェルネスに焦点を当てたプロフェッショナルサービスを提供しています。

データ準備とデータウェルネスサービスについての詳細情報は、[データの準備]()を参照してください。
