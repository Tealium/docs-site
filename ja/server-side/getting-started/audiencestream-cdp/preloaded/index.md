AudienceStreamとEventStreamには、多くのお客様が必要とする一連のプリロードされた訪問者と訪問属性が含まれています。各属性は、訪問者や訪問について有用な情報を決定するのに役立つ統計情報で事前に設定されています。

![](https://docs.tealium.com/images/server-side/getting-started/audiencestream-cdp/preloaded-attributes.png)

これらの属性を自分自身の属性の出発点として使用することができます。プリロードされた属性は変更できないため、プリロードされた属性を複製し、そのコピーを自分のニーズに合わせてカスタマイズします。

これらの属性についての詳細情報は、[プリロードされた属性](https://docs.tealium.com/preloaded-attributes/)を参照してください。
