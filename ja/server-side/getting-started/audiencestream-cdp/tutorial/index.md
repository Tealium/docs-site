## 必要条件

このガイドを完了するために必要なもの：

* Tealiumサーバーサイドのログイン情報
* 既存のTealium EventStreamアカウント
* アクティブ化されたAudienceStreamアカウント
* サーバーサイドAPIを提供するベンダー（オプション）

## Tealium AudienceStreamとは何ですか？

![](/images/tutorials/audiencestream-customer-evolution.jpg)

Tealium AudienceStreamは、ID解決、データエンリッチメント機能、および強力なオーディエンス管理を提供するカスタマーデータプラットフォーム（CDP）です。これは、最も価値のあるファーストパーティデータを表す訪問プロファイルを作成し、共有行動によってセグメント化されたオーディエンスを作成し、リアルタイムのコネクタを介してマーケティング技術スタックに燃料を供給するために使用されます。

Tealium AudienceStreamは以下の利点を提供します：

* ビジタースティッチング技術は、一貫した関連性のある顧客体験を提供するために、顧客プロファイルの統一されたビューを作成します。
* リアルタイムの顧客データ信号に基づいてアクションをトリガーするためのベンダーとのAPI統合。

## ユーザーインターフェースを探索する

&lt;https://my.tealiumiq.com/&gt;にアクセスし、Tealiumにログインしてインターフェースを探索します。

AudienceStreamは以下のエリアに分かれています：

* **ダッシュボード** - 訪問、オーディエンス、バッジ活動の概要。

* **ディスカバー** - 訪問の属性を探索して、ライブと履歴のセグメントを発見します。

* **属性（訪問/訪問）** - AudienceStream構成の基礎となる訪問/訪問の属性を管理します。これらは、お客様とその活動を説明します。

* **オーディエンス** - 属性のセットを共有する訪問プロファイルの集まりであるオーディエンスを作成します。これらは、あなたが構成する訪問データマッピングを使用してベンダーAPIをトリガーするサーバーサイドのアクションを引き起こします。

* **コネクタ** - AudienceStream CDPの中核となるベンダー統合を管理します。これらのサーバーサイドのアクションは、あなたが構成する訪問データマッピングを使用してベンダーAPIをトリガーします。
