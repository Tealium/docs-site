### はじめに


![](/images/icons/icon-forward.svg)

#### カスタマーデータハブ

ここから始めて、その仕組みと使い方を学びましょう。



![](/images/icons/icon-product-eventstream.svg)

#### イベントストリームAPIハブ

オンラインおよびオフラインのデータ収集を管理し、データ品質を確保し、サーバーサイドの分析およびマーケティングツールと統合します。



![](/images/icons/icon-cubes.svg)

#### 属性

イベントと訪問の基本的なデータ要素を管理します。


### データ収集


![](/images/icons/icon-arrow-right-into.svg)

#### データソース

ウェブサイト、モバイルアプリ、またはカスタムアプリケーションからTealiumにデータを送信します。



![](/images/icons/icon-product-data-connect.svg)

#### データコネクト

イベントと訪問のデータをTealiumに統合し、データを豊かにします。



![](/images/icons/icon-file-arrow-up.svg)

#### ファイルインポート

オフラインおよびオンラインチャネルから顧客データをインポートします。



![](/images/icons/icon-cubes-check.svg)

#### 仕様

受信データの品質を整理し、検証します。



![](/images/icons/icon-arrows-merge.svg)

#### フィード

最も重要なイベントタイプのフィードを定義します。


### 統合


![](/images/icons/icon-connector.svg)

#### コネクタ

コネクタを使用して、イベントと訪問のデータをお好みのベンダーAPIと統合します。



![](/images/icons/icon-code.svg)

#### Webhooks

Webhookコネクタを使用してカスタム統合を構築します。
