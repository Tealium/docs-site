Live Eventsは未知のイベントを検出し、イベント詳細ビューから直接カスタム仕様を作成できます。これを行うと、イベントの属性が自動的に**イベント仕様の追加**ウィンドウに含まれます。

## 仕組み

カスタム仕様でもマーケットプレイスからの仕様でも、**Live Events**画面のイベント詳細ビューから作成できます。この例では、`search`イベントで終わったところから始めます。

以下の手順で行います：

1. イベント詳細が表示されている状態で、**イベント仕様を作成**をクリックします。
    * イベント仕様はイベントの属性で事前に入力されます。
    * 未知の属性がリストされた確認ダイアログが表示されます。**未知の属性を除外**をクリックして、これらの属性を新しいイベント仕様から除外します。
1. **ルール**の下で、イベント仕様に使用するイベント名を入力します。例えば、`search`。
1. **作成**をクリックします。

イベント仕様を作成したので、**Live Events**画面でデータ品質を観察できます。
