パブリッシャー

この記事では、パブリッシャーのデータレイヤーの定義を提供します。

データレイヤーの属性

変数 説明 例 タイプ
click_category クリックトラッキングイベント中、クリックされた要素のカテゴリー footer 文字列
click_label クリックトラッキングイベント中、クリックされた要素の名前 Submit 文字列
click_type クリックトラッキングイベント中、異なる要素を区別するためのタイプ button 文字列
content_author コンテンツの著者の名前 John Smith 文字列
content_id ページコンテンツの固有ID 38121 文字列
content_publish_date ページコンテンツの公開日（YYYY/MM/DD形式） 2015/02/09 文字列
content_title ページ上のコンテンツのタイトル Top 7 Outdoor Sleeping Bags 文字列
country_code サイトの2文字の国コード us 文字列
customer_city 顧客の居住都市 San Diego 文字列
customer_country 顧客の居住国 United States 文字列
customer_email 顧客のメールアドレス johnsmith@example.com 文字列
customer_first_name 顧客の名前 John 文字列
customer_id 顧客の固有ID（customer_emailでない場合） 8237572 文字列
customer_is_logged_in 現在のユーザーがログインしているかどうかを示すフラグ 1 文字列
customer_last_name 顧客の姓 Smith 文字列
customer_postal_code 顧客の郵便番号 92101 文字列
customer_state 顧客の居住州/省 CA 文字列
customer_type 登録ユーザーとゲストを区別する guest 文字列
language_code サイトの2文字の言語コード en 文字列
page_friendly_url ページの正規URL（ブラウザの場所バーのURLとは異なる場合があります） /travel/national_parks 文字列
page_name ページのユーザーフレンドリーな名前 Homepage 文字列
page_number ページネーションされたコンテンツやギャラリーの場合、表示中の現在のページ番号 2 文字列
page_type ページまたはテンプレートのタイプを識別するためのTealium変数 content 文字列
search_keyword ユーザーが入力した検索テキスト Napa valley 文字列
search_results 返された検索結果の数 42 文字列
site_display_format 表示中のサイトの表示形式 mobile 文字列
site_section サイトのトップレベルセクション Travel 文字列
site_subsection1 ページ階層の1レベルカテゴリー National Parks 文字列
site_subsection2 ページ階層の2レベルカテゴリー California 文字列
site_subsection3 ページ階層の3レベルカテゴリー Yosemite 文字列
site_subsection4 ページ階層の4レベルカテゴリー Camping 文字列
slide_title ギャラリーページの場合、表示中の現在のスライドのタイトル #2 Yellowstone 文字列
social_network ソーシャルシェアイベント中に使用されたソーシャルネットワークの名前 facebook 文字列

ホームページ

イベント名 説明 サンプルURL
page_view ホームページ。 http://www.example.com/

サンプル:

{
    "country_code"           : "us",
    "customer_email"         : "johnsmith@example.com",
    "customer_first_name"    : "John",
    "customer_id"            : "8237572",
    "customer_is_logged_in"  : "1",
    "customer_last_name"     : "Smith",
    "customer_type"          : "registered",
    "language_code"          : "en",
    "page_friendly_url"      : "/home",
    "page_name"              : "Homepage",
    "page_type"              : "home",
    "site_display_format"    : "desktop",
    "site_section"           : "Home",
    "tealium_event"          : "page_view"
}

カスタムクリック

イベント名 説明
custom_click ページ上の特定のクリックインタラクションのトラッキング。

サンプル:

    "tealium_event"  : "custom_click"
}
