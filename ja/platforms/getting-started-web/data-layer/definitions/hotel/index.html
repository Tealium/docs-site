ホテル

この記事では、ホテルのデータレイヤーの定義を提供します。

データレイヤーの属性

変数 説明 例 タイプ
avg_room_rate 各部屋の平均料金の配列 ["100.25"] 配列
cancellation_id 予約がキャンセルされたときに与えられるID 12254875 文字列
check_in ユーザーの到着日 02/18/2017 文字列
check_out ユーザーの出発日 02/24/2017 文字列
check_reservation 予約へのアクセス方法（確認または情報） information 文字列
currency 料金の通貨 USD 文字列
customer_city 予約者の居住都市 San Diego 文字列
customer_country 予約者の居住国 USA 文字列
customer_email 予約者のメールアドレス example@example.com 文字列
customer_first_name 予約者の名前 Tealium 文字列
customer_id_type ロイヤリティサイトのログインに使用されるIDのタイプ（メールまたはロイヤリティID） loyalty id 文字列
customer_last_name 予約者の姓 Tealium 文字列
customer_postal_code 予約者の郵便番号 92121 文字列
customer_state 予約者の州 CA 文字列
days_till_checkin 今日とチェックイン日の間の日数 45 文字列
displayed_properties Select Hotelページに表示されるすべてのプロパティのプロパティIDの配列 ["345","4567"] 配列
displayed_rate_amount Select Roomページで表示される料金 150.00 文字列
displayed_rate_amounts Select Hotelページに表示されるすべてのプロパティの料金の配列 ["100.25","150.00","340.00"] 配列
displayed_rate_codes Select Hotelページに表示されるすべてのプロパティの料金コードの配列 ["economy","economy","luxury"] 配列
displayed_room_rates Select Roomページで部屋タイプごとにグループ化された表示料金の配列 ["340.00","500.00","650.00"] 配列
displayed_room_types Select Roomページで表示される部屋タイプの配列 ["2 Queen Beds","Ocean View","Ocean Front"] 配列
error_message ユーザーに表示されるエラーテキスト Room no longer available 文字列
favorite_property_id 「お気に入り」に追加されたプロパティのプロパティID 123 文字列
filter_amenities 適用された選択の配列 ["pool","microwave"] 配列
filter_distance イベントからのフィルター距離 20 文字列
filter_hotel_type ユーザーによってフィルタリングされたホテルタイプの配列 ["economy","luxury"] 配列
filter_max_rate 適用される料金 150 文字列
filter_rate_change フィルターの変更 AAA to 20% Off 文字列
filter_ta_rating ユーザーによってフィルタリングされたTravel Advisorの評価の配列 ["3","4","5"] 配列
language_code 言語コード us 文字列
link_category クリックトラッキングイベント時にクリックされた要素のカテゴリー Navigation 文字列
link_name クリックトラッキングイベント時にクリックされた要素の名前 properties 文字列
login_status ユーザーがログインしているかログアウトしているか logged in 文字列
loyalty_id ロイヤリティ会員番号 99568784521 文字列
loyalty_point_balance ユーザーのロイヤリティアカウントの現在のポイント残高 1000 文字列
loyalty_tier メンバーシップステータスレベル Silver 文字列
modify_stay ステイが変更されるたびに構成されるイベントトラッキング true 文字列
num_of_adults 大人の総数 2 文字列
num_of_children 子供の総数 3 文字列
num_of_nights 泊数の総数 5 文字列
num_of_rooms 部屋の総数 1 文字列
occupancy_detail 検索された各部屋の総宿泊人数の配列 ["4","2"] 配列
order_cancellation_policy 注文ごとの各部屋のキャンセルポリシーの配列 ["24hours","5days"] 配列
order_corp_id 注文に添付された法人ID（ある場合） 987646 文字列
order_discount_amount 注文レベルの割引額 100.00 文字列
order_iata 注文に添付された国際航空運送協会番号（旅行代理店番号など） 646879 文字列
order_id 注文の一意のID（注文確認ページでのみ入力されるべき） 1234 文字列
order_payment_method 支払い方法 visa 文字列
order_payment_type 標準の収益またはポイント revenue 文字列
order_points_used 注文ごとの各部屋の合計ポイント金額の配列 ["1000","5000"] 配列
order_postal_code 予約されたプロパティの郵便番号 00267 文字列
order_promo_code 注文に添付されたプロモーションコード（ある場合） SALE 文字列
order_rate_code 注文ごとの各部屋の予約された料金コードの配列 ["AAA",""] 配列
order_room_occupancy 注文ごとの各部屋の総宿泊人数の配列 ["5","2"] 配列
order_room_type 注文ごとに予約された各部屋の部屋タイプの配列 ["double","ocean view"] 配列
order_special_requests 「特別リクエスト」フィールドに入力されたフリーテキスト（部屋ごと） No Feathers 文字列
order_subtotal 商品または注文レベルの割引を含む合計小計（税金や手数料を除く） 1000.00 文字列
order_tax_amount 税金と手数料の合計額 75.00 文字列
order_total 注文の合計金額 1175.00 文字列
order_total_occupancy 予約時の注文の総宿泊人数 5 文字列
page_joined ロイヤリティに参加するページ - ページ名 homepage 文字列
page_name ページ名を識別するためのTealium変数 homepage 文字列
page_type ホーム、プロパティ、確認などの高レベルのページタイプ home 文字列
promo_registration_page プロモーションの登録が行われたページ offers 文字列
property_brand プロパティのブランド名 ["economy brand"] 配列
property_city プロパティがある都市 ["Chula Vista"] 配列
property_country プロパティがある国 ["USA"] 配列
property_id プロパティ番号の配列 ["354498765467"] 配列
property_name プロパティ名の配列 ["hotel Chula Vista"] 配列
property_position 選択されたプロパティの最高位置 ["1"] 配列
property_postal_code プロパティの郵便番号の配列 ["92121"] 配列
property_state プロパティがある州の配列 ["CA"] 配列
proximity 都市の中心とプロパティの位置の距離（マイル）の配列 ["24"] 配列
registration_id 特別オファーやプロモーションに登録するためにユーザーが入力したID（メールまたはロイヤリティID） example@example.com 文字列
search_city 検索された目的地の都市 Chula Vista 文字列
search_country 検索された目的地の国 USA 文字列
search_full Googleによって提供される目的地の詳細な説明 Chula Vista, California, United States 文字列
search_rate 予約ウィジェットの「Select Rate」ドロップダウンメニューで選択された料金 Corporate 文字列
search_rate_value 予約ウィジェットの「Select Rate」ドロップダウンメニューでのプロモーションと法人の選択に入力された値 Tealium 文字列
search_results 検索結果の数 22 文字列
search_state 検索された目的地の州 WA 文字列
search_type Googleによって定義された空港、地域、国立公園などの目的地のタイプ airport 文字列
select_room_position Select HotelページからSelect Roomページに表示された料金の最高位置 1 文字列
selected_rate_price ユーザーがSelect Roomページで選択した部屋の料金の配列 ["100.00","150.00"] 配列
selected_rate_type ユーザーがSelect Roomページで選択した部屋の料金タイプの配列 ["Best Rate","loyalty","AAA"] 配列
selected_room_type ユーザーがSelect Roomページで選択した部屋の部屋タイプの配列 ["2 double beds","Ocean View"] 配列
show_available 利用可能なホテルのみを表示する構成 true 文字列
site ユーザーがいるサイト economy site 文字列
site_section サイトの上位セクション offers 文字列
social_channel イベントで情報がソーシャルメディアで共有されたときに使用 facebook 文字列
sort_by ソート順（距離、低から高、高から低）の選択 distance 文字列
sort_order Select Roomページで表示される部屋タイプ、料金コード、料金、通貨の配列 ["room type","distance"] 配列
video_action ビデオで現在実行中のアクション（再生、再生中、一時停止、停止、完了） play 文字列
video_content_type 閲覧したビデオのタイプ Property Video 文字列
video_name 閲覧したビデオの名前 Video Name 文字列
video_segments 閲覧したビデオのパーセント 75 文字列
