## Android/iOS 用 API

以下のリソースは、Android および iOS 用の Tealium API リファレンスへのリンクを提供します。Xamarin でモバイルアプリケーションを構築する場合は、iOS および Android ライブラリへのバインディングを使用してください。

*   [Tealium for iOS](/ja/platforms/ios-objective-c/api/)
*   [Tealium for Android](/ja/platforms/android-java/api)

## C# 用 API

また、Tealium Customer Data Hub サービス（Tealium AudienceStream、Tealium EventStream、DataAccess）のみを使用する場合は、[Tealium for C#](/ja/platforms/csharp/api) ライブラリを Xamarin プロジェクトのネイティブコードとして使用することができます。
