Tealium + Facebookリード広告ガイド

この記事では、LeadsBridgeを使用してFacebookリード広告をTealiumに統合する方法について説明します。

前提条件

  • Tealium EventStream API Hub および/または Tealium AudienceStream CDP
  • LeadsBridgeまたはZapierのアカウント
  • Facebookビジネスマネージャー - リードアクセス
    詳細については、リアルタイムリードアクセスの構成を参照してください。

仕組み

Facebookリード広告は、ユーザーをランディングページにリダイレクトせずに広告内からユーザーの情報を収集するリードジェネレーション広告の一種です。Facebookリード広告は、広告内のフォームを通じてユーザーの情報を収集し、収集プロセスをより速く、ユーザーにとってより便利にします。収集した情報は、メールマーケティングやダイレクトセールスのアウトリーチなどのフォローアップ活動に使用できます。

Facebookリード広告からの顧客情報をTealium EventStreamまたはAudienceStreamに渡すには、LeadsBridgeまたはZapierのWebhookを使用する必要があります。このドキュメントでは、LeadsBridgeのWebhookの構成方法について説明しますが、同様の構成でZapierのWebhookも使用できます。

Webhookの構成には、tealium_account、tealium_profile、tealium_datasource、およびFacebookリード広告からのLead IDの値を含めてください。

Facebookリードデータソースの追加

  1. FacebookリードデータソースをTealium Customer Data Hubプロファイルに追加するには、データソースの作成の指示に従ってください。
  2. Facebookリードデータソースを作成した後、データソースに接続の指示に従って、Data Source KeyとData Source URLを見つけます。
  3. データソースを追加し、接続した後、プロファイルを保存して公開します。

データソースURL

FacebookリードデータソースをTealium Customer Data Hubプロファイルに追加すると、次の形式でFacebookリード広告の構成で使用するためのユニークなURLが生成されます：

https://collect.tealiumiq.com/integration/event/ACCOUNT/PROFILE/DATA_SOURCE_KEY

Facebookビジネスマネージャーを使用してFacebookリードを構成する

  1. Facebookアカウントにログインし、ビジネスマネージャーとインスタントフォームを選択に移動します。
  2. Tealiumにデータを送信するために使用したいフォームを選択します。
  3. CRM構成を選択し、Webhookを検索します。
  4. あなたの統合URLとLeadsBridgeを使用するようにWebhookを構成します。
  5. Facebookのドキュメンテーションに記載されているFacebookにCRMシステムを接続するの手順を完了します。

LeadsBridgeのWebhookをFacebookリード広告と統合する

  1. Facebookリード広告を構成します。

  2. LeadsBridgeアカウントで、メインメニューからFacebook Suite > Lead Ads Bridgesに移動します。

  3. +新しいブリッジを作成をクリックします。

  4. ソースを選択の下で、Facebook Lead Adsを検索し、ドロップダウンリストからソースを選択します。

  5. 宛先を選択の下で、Webhookを検索し、ドロップダウンリストから宛先を選択します。

  6. LeadsBridgeは、高度なエディターを使用するように促します。複雑なブリッジに連れて行ってをクリックします。

  7. Facebookリード広告の始めましょう画面で、次の構成を行います。完了したら、次へをクリックします。

    • ブリッジの名前 - ブリッジの名前を入力します。
    • ソースを選択 - ソースの選択としてFacebookリード広告をそのままにします。
    • 宛先を選択 - Webhooksを入力し、Webhooksのドロップダウンリストから新しい宛先を作成を選択します。

  8. ソース画面で、次のフィールドを構成します。完了したら、次へをクリックします。

    • ユーザー - ユーザーを選択します。+Facebookと接続オプションを使用して認証する必要があるかもしれません。
    • Facebookページ - ドロップダウンリストからFacebookページを選択します。Facebookアカウントの認証が必要になるかもしれません。
    • Facebookフォーム - ドロップダウンリストからFacebookフォームを選択します。Facebookアカウントの認証が必要になるかもしれません。

  9. 宛先 > 宛先統合の構成画面で、次のフィールドを構成します。完了したら、次へをクリックします。

    • 統合の名前 - 統合の名前を入力します。
    • 方法 - POSTを選択します。
    • URL - TealiumデータソースURLを入力します。
    • アクセスシークレット - 空白のままにします。
    • タイムアウト（秒） - Webhookのタイムアウトを秒単位で入力します。30秒を推奨します。
    • カスタムフィールド - 次のフィールドを入力します：tealium_account、tealium_profile、tealium_datasource、およびFacebookリード広告の送信からの他のフィールド、例えばlead_id、email、tealium_event。

  10. 宛先 > 受信リードを送信する場所を選択画面で、次の構成を行います。完了したら、次へをクリックします。

    • 宛先とどのアクションフィールドは、前の画面からの情報で完成します。
    • （オプション）各リードのメールレシートを受け取りたい場合は、**アクティブ？**オプションを有効にします。受信者フィールドにメールアドレスを入力し、件名フィールドにメールの件名を入力します。

  11. フィールドマッピング画面で、Tealiumにデータをトリガーするために必要な値をドロップダウンリストから選択します。また、次のマッピングフィールドを構成します：tealium_account、tealium_profile、tealium_datasource。

  12. 次へをクリックします。

  13. （オプション）テスト画面で、指定したフィールドの値を入力し、今すぐテストをクリックします。

    Facebookリード広告テストツールを使用して構成をテストすることもできます。

  14. LeadsBridgeのアクセス許可を受け入れ、保存して公開をクリックします。

イベントと属性

すべての受信Facebookリードイベント属性は自動的にfblead_でプレフィックスされます。例えば、Currentsがidを送信すると、対応するEventStreamイベント属性はfblead_id（すべて小文字）になります。

キー タイプ 例
’tealium_account' 文字列 sbx-facebook
tealium_profile 文字列 main
tealium_datasource 文字列 r9zxm1
tealium_event 文字列 facebook_add_event
tealium_visitor_id 文字列 cb7425dd8b1742c68851b049f22a0154
fblead_email 文字列 facebook@fb.com
fblead_full_name 文字列 Bob White
fblead_address 文字列 25 First Street
fblead_other_field 文字列 Other fields
