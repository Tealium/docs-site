更新と修正

    • EventStreamのサポートを追加しました。
    • /amc/partner/instancesエンドポイントを非推奨とし、/amc/instancesに移行しました。
  • Amazon Ads Conversionsコネクタを更新し、追加のユーザーマッチキーをサポートしました。
  • Amazon Advertising DSPコネクタを更新し、オーディエンス作成のための国コードをサポートしました。
  • Firebase Cloud Messagingコネクタを更新し、テンプレートレンダリングの問題を修正しました。
  • LinkedIn Conversionsコネクタを更新し、メールパラメータの送信エラーを修正しました。追加のユーザー情報（会社、役職、国コードなど）を送信する場合、メールまたはUUIDが提供されていても、ユーザーの名前と姓が必要です。
  • Reddit Conversionsコネクタを更新し、transaction_idを削除し、conversion_idと製品パラメータをConversion Eventマッピングセクションに追加しました。
  • Google Analytics 4 Measurement Protocolコネクタを更新し、page_title、page_location、page_referrerの文字制限を削除し、ユーザー提供のデータと同意状態をサポートしました。
  • Yandex Metricaコネクタを以下の変更で更新しました：
    • 構成：Counter IDの値がコネクタ構成の一部になりました。
    • Yandex Client ID (_ym_uid)がマッピング可能で、必須パラメータになりました。
    • Productsパラメータは、製品ID（キー）を表す配列と、各製品の数量（値）を表す配列の2つの配列になりました。
    • Send Order Event：日付型の属性が必要になりました。
    • Send Ecommerce Event：Currency codeがマッピング可能になり、Action FieldはPurchase Fieldsになり、必須ではなくなりました。
