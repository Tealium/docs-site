## API情報

このコネクタは以下のベンダーAPIを使用します：

* API名：Teads変換API
* APIバージョン：v1
* APIエンドポイント：`https://ca.teads.tv/`

## 構成

コネクタマーケットプレイスに移動し、新しいコネクタを追加します。コネクタの追加方法の一般的な指示については、[コネクタについて]()を参照してください。

コネクタを追加した後、以下の構成を行います：
* **変換APIトークン**  
  * 変換APIトークンを生成するには、構成パネルに移動し、**変換APIトークン &gt; 変換APIトークン生成**をクリックします。

## アクション

| アクション名 | AudienceStream | EventStream |
| --- | :---: | :---: |
| 変換イベントの送信 | ✗ | ✓ |

アクションの名前を入力し、ドロップダウンメニューからアクションタイプを選択します。

次のセクションでは、各アクションのパラメータとオプションの構成方法について説明します。

### 変換イベントの送信

#### パラメータ

| **パラメータ** | **説明** |
| --- | --- |
| イベントID (`auctid`) | イベントの一意の識別子。`URL`（`auctid`パラメータ）またはクッキー（`tfpai`）から取得できます。 |
| 購入者ピクセルID | ウェブサイトにインストールされたピクセルの`ID`。 |
| イベントソースURL | ユーザーが訪れたページの`URL`。 |
| 変換タイプ | 変換イベントのタイプ：`pageView`、`AddToCart`、`AddToWishlist`、`CompleteRegistration`、`InitiateCheckout`、`Lead`、`Purchase`、`Search`、`ViewContent`。デフォルトは`pageView`イベントです。 |
| イベント時間 | イベントのタイムスタンプ（エポック秒）。 |

#### 変換パラメータ

| **パラメータ** | **説明** |
| --- | --- |
| 名前 | （文字列）名前は最大`100`文字まで。 |
| 価格 | （小数）指定された通貨パラメータの単位での金額価格。 |
| 通貨 | （文字列）通貨を表す3文字の`ISO 4217`通貨コード。 |

#### ファーストパーティクッキー

| **パラメータ** | **説明** |
| --- | --- |
| `tfpai` | `auctid`値を含みます。 |
| `tfpvi` | ユーザー識別子を含みます。 |
| `tfpsi` | ユーザーセッションを計算するために使用されるユーザーセッション`ID`を含みます。 |
