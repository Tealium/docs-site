## アクションとパラメータ

**次へ**をクリックするか、**アクション**タブに移動します。ここでコネクタのアクションを構成します。

このセクションでは、各アクションのパラメータとオプションの構成方法について説明します。

### ランディングページイベントの送信

| **パラメータ** | **説明** |
| --- | --- |
| キャンペーンID | (必須) Impactが提供する一意の整数キャンペーン識別子。 |
| ページURL | (必須) フルランディングページURL、例：`https://www.example.com?utm_source=google&amp;utm_medium=cpc`. |
| カスタムプロファイルID | (必須) 広告主独自のユーザー固有の値、例：`cookie value`. |
| ユーザーエージェント | (必須) ブラウザとオペレーティングシステムを識別する文字列。 |
| IPアドレス | (必須) 顧客のIPアドレスを表す文字列。 |
| リファリングURL | クリックが発生したページのURL。 |
| 顧客ID | お客様のプラットフォームが顧客アカウントに割り当てる一意の英数字の顧客識別子。個人を特定できる情報は使用しないでください。 |
| 顧客のメール | 顧客のメールアドレスのSHA1ハッシュ文字列。 |
| イベント日付 | `NOW`の文字列値または&lt;a href=&#34;https://en.wikipedia.org/wiki/ISO_8601&#34; target=&#34;_blank&#34;&gt;ISO 8601&lt;/a&gt;または`dd-MMM-yyyy HH:mm:ss z`形式の日付を使用します。デフォルトでは`NOW`が使用されます。 |
| 顧客のメールはすでにハッシュ化されている | 顧客のメールがすでにハッシュ化されている場合は、このボックスをチェックします。 |
