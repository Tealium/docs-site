## API情報

このコネクタは、以下のベンダーAPIを使用します：

* API名：Mailgun API
* APIバージョン：v3
* APIエンドポイント：`https://api.mailgun.net`
* ドキュメンテーション：[Mailgun API](https://documentation.mailgun.com/en/latest/)

## コネクタアクション

| アクション名 | AudienceStream | EventStream |
| --- | :---: | :---: |
| メール送信 | ✓ | ✓ |
| リスト内のメンバーをアップサート | ✓ | ✓ |

## 構成の構成

コネクタマーケットプレイスに移動し、新しいコネクタを追加します。コネクタの追加方法の一般的な指示については、[コネクタについて]()を参照してください。

コネクタを追加した後、以下の構成を構成します：

* **Mailgunドメイン**：必須。適切な地域ベースのドメインを選択します。
* **APIキー**：必須。プライベートAPIキーはMailgunダッシュボードで見つけることができます。

## アクション構成 - パラメータとオプション

アクションの名前を入力し、ドロップダウンメニューからアクションタイプを選択します。

以下のセクションでは、各アクションのパラメータとオプションの構成方法について説明します。

### アクション - メール送信

#### パラメータ

| **パラメータ** | **説明** |
| --- | --- |
| 送信ドメイン | (必須) Mailgunアカウントのドメイン。 |
| 差出人 | (必須) 送信者のメールアドレス。 |
| 宛先 | (必須) 受信者のメールアドレス。  |
| 件名 | メールの件名。 |
| テキスト | メッセージの本文。 |
| CC | CCするメールアドレス。 |
| BCC |BCCするメールアドレス。 |
| テンプレート名 | Mailgunのメール送信テンプレート名。 |
| テンプレートテキスト | メッセージのテキストをテンプレートを通じてレンダリングしたい場合は、このパラメータを**はい**に構成します。 |
| テンプレートバージョン | メールを特定のテンプレートバージョンに送信します。 |
| テンプレート変数 | テンプレート変数展開の入力として使用されるJSONエンコードされた辞書を作成するための属性をマップします。詳細については、[Mailgun APIリファレンス](https://documentation.mailgun.com/en/latest/api-sending.html#sending)を参照してください。 |

### アクション - リスト内のメンバーをアップサート

#### バッチ制限

このアクションは、ベンダーへの大量データ転送をサポートするためにバッチリクエストを使用します。詳細については、[バッチアクション]()を参照してください。リクエストは、以下のいずれかの閾値が満たされるか、プロファイルが公開されるまでキューに入れられます：

* リクエストの最大数：1000
* 最古のリクエストからの最大時間：60分
* リクエストの最大サイズ：1 MB

#### パラメータ

| **パラメータ** | **説明** |
| --- | --- |
| アドレス | (必須) リストのメールアドレス。 |
| メンバーアドレス | (必須) リストに追加するメールアドレス。 |
| メンバー名 | 追加するメールアドレスの名前。 |
| 登録 | &lt;ul&gt;&lt;li&gt;**はい**：新しいメンバーを登録します。&lt;/li&gt;&lt;li&gt;**いいえ**：新しいメンバーを登録しません。&lt;/li&gt;&lt;/ul&gt; |
| アップサート | &lt;ul&gt;&lt;li&gt;**はい**：リストに存在する場合はリストメンバーを更新します。&lt;/li&gt;&lt;li&gt;**いいえ**：重複していてもリストメンバーを追加します。&lt;/li&gt;&lt;/ul&gt; |
| 変数 | 追加のメンバーパラメータを渡すための属性をマップします。詳細については、[Mailgun APIリファレンス](https://documentation.mailgun.com/en/latest/api-sending.html#sending)を参照してください。 |
