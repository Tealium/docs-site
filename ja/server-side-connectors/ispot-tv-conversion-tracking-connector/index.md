## 動作原理

iSpot TV コンバージョンアナリティクスは、トラッキングコードを使用して、1700万台以上のテレビの広告露出をあなたのファーストパーティのウェブサイトやアプリデータと連携します。各テレビはホーム庭のネットワークに接続されており、iSpotがホーム庭のIPアドレスにアクセスできるようになります。これにより、iSpotはテレビ広告の露出をIPアドレスに関連付けられたデータセットと連携でき、TVクリエイティブ、メディア、頻度が販売やその他の重要なKPIをどのように推進するかについての強力なクローズドループ分析を実現します。全プロセスは、個人を特定できる情報（PII）に準拠した方法で行われます。

## コネクタのヒント

タグとコネクタを使用したハイブリッド構成を推奨します。この構成により、データの耐久性が最大化され、サーバーサイドのシグナルとのイベントマッチングが改善され、リアルタイムのクライアントサイドシグナルを保持しながらデータフローの連続性が維持されます。

## 構成の構成

コネクタマーケットプレイスにアクセスし、新しいコネクタを追加します。コネクタを追加する一般的な手順については、[Connector Overview](https://docs.tealium.com/about-connectors/)の記事を読んでください。

コネクタを追加した後、以下の構成を構成します：

* **サイトID**
  * トラッキングコードには、各ウェブサイトまたはアプリごとに一意のサイトIDが必要で、iSpotによって提供されます。
  * サイトIDの形式は `TC-1234-1` です。ここで：
    * `TC` はトラッキングコードを表します。
    * `1234` はクライアントのアカウントIDです。
    * `-1` はサイト識別子で、サイトごとに増加します。

## アクション構成 - パラメータとオプション

**次へ** をクリックするか、**アクション** タブに進みます。ここでコネクタアクションを構成します。

このセクションでは、各アクションのパラメータとオプションの構成方法について説明します。

### アクション - Send Pixel

#### パラメータ

| **パラメータ** | **説明**        |
|:--------------|:-----------------------|
| IPアドレス    | <ul><li>IPアドレスはIPv4のみです。</li><li>例： `192.168.0.1`</li><li>Tealiumは `X-Forwarded-For` リクエストヘッダーで値を渡します。</li></ul>           |
| ユーザーエージェント    | <ul><li>元のクライアントデバイスのユーザーエージェントです。</li><li>Tealiumはリクエストヘッダーで `User-Agent` を渡します。</li></ul>          |
| ユーザーエージェント    | <ul><li>元のクライアントデバイスのユーザーエージェントに構成します。</li><li>Tealiumはリクエストヘッダーで `User-Agent` を渡します。</li></ul>          |
| クッキー       | <ul><li>iSpotのクッキーは、Cookieリクエストヘッダーの一部として送信されます。</li><li>このクッキーの値は、サービスからのJSONレスポンスを通じて元のクライアントサイドブラウザから導出できます。</li><li>例： `https://ps.ispot.tv/v2/your-site-id.gif`</li></ul>     |
| リファラー      | <ul><li>元の訪問のリファラーで、リファラーリクエストヘッダーで送信されます。</li></ul>         |
| イベント時間    | <ul><li>クライアント訪問とサーバーサイドリクエストの時間差が数分以上ある場合、イベント時間をクライアント訪問の元のUTC時間で `yyyy-MM-dd hh:mm:ss` 形式に構成します。</li><li>イベント時間を指定しない場合、コネクタは実行時に生成します。</li></ul> |
| カスタムデータ   | <ul><li>iSpotトラッキングコードは、ビジネス目標のためのカスタムデータの渡しをサポートします。</li><li>このセクションで個々の属性を渡し、コネクタはそれらをURLエンコードされたJSONオブジェクトに組み合わせます。</li></ul>                                                                                         |
