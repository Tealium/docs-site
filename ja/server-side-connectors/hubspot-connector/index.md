## 構成

コネクタマーケットプレイスにアクセスし、新しいコネクタを追加します。コネクタを追加する一般的な手順については、[コネクタについて](https://docs.tealium.com/about-connectors/)を参照してください。

コネクタを追加した後、以下の構成を構成します：

* **プライベートアプリアクセストークン**：（必須）プライベートアプリアクセストークンを提供してください。この値は、HubSpotのデータを取得するためのアクションパラメータフィールドに正しく入力する必要があります。詳細については、[HubSpot: プライベートアプリ](https://developers.hubspot.com/docs/api/private-apps)を参照してください。

## アクション

| アクション名 | AudienceStream | EventStream |
| --- | :---: | :---: |
| コンタクトの作成（リアルタイム） | ✓ | ✗ |
| コンタクトの作成（バッチ処理） | ✓ | ✗ |
| コンタクトの更新（リアルタイム） | ✓ | ✗ |
| コンタクトの更新（バッチ処理） | ✓ | ✗ |
| コンタクトをリストに追加 | ✓ | ✗ |
| リストからコンタクトを削除 | ✓ | ✗ |

アクションの名前を入力し、**アクションタイプ**を選択します。

以下のセクションでは、各アクションのパラメータとオプションの構成方法について説明します。

### コンタクトの作成（リアルタイム）

|**パラメータ**| **説明**|
|---| ---|
|コンタクトプロパティ| <ul><li>（必須）コンタクトのメールアドレス。</li><li>リストからHubSpotフィールドを選択します。</li></ul>|

### コンタクトの作成（バッチ処理）

#### バッチ制限

このアクションは、バッチリクエストを使用してベンダーへの大量データ転送をサポートします。以下のいずれかの閾値に達するまでリクエストがキューに入ります：

* 最大リクエスト数：10
* 最古のリクエストからの最大時間：5分
* リクエストの最大サイズ：1 MB
