## API情報

このコネクタは以下のベンダーAPIを使用します：

* API名：TikTok HTTP API
* APIバージョン：v1.3
* APIエンドポイント：`https://business-api.tiktok.com/open_api/v1.3`
* ドキュメント：[TikTok Audience segment](https://ads.tiktok.com/marketing_api/docs?id=1736874036878337)

## バッチ制限

このコネクタは、ベンダーへの大量データ転送をサポートするためにバッチリクエストを使用します。以下のいずれかの閾値に達するまでリクエストがキューに入れられます：

* 最大リクエスト数：1000
* 最古のリクエストからの最大時間：30分
* リクエストの最大サイズ：1 MB

## 構成

コネクタマーケットプレイスにアクセスして新しいコネクタを追加します。コネクタを追加する一般的な手順については、[コネクタについて](https://docs.tealium.com/about-connectors/)を参照してください。

コネクタを追加した後、以下の構成を構成します：

* **広告主ID**：（必須）TikTok広告マネージャーからの広告アカウントID。詳細については、[TikTok: 広告アカウントIDの検索方法](https://ads.tiktok.com/help/article/find-your-ad-account-id?lang=en)を参照してください。

**接続を確立**をクリックして、TikTokへの接続をテストします。

このコネクタのためのオーディエンスを作成するには、以下の手順に従います：

1. **オーディエンスを作成**をクリックします。
1. **カスタムオーディエンス名**の下に、このコネクタが使用するオーディエンス名を入力します。
1. **暗号化タイプ**のドロップダウンリストから、オーディエンスの作成時に使用する暗号化タイプを選択します。
1. **作成**をクリックし、**完了**をクリックします。

オーディエンスを削除するには、以下の手順に従います：

1. **オーディエンスを削除**をクリックします。
1. **オーディエンスID**のドロップダウンリストから、削除するオーディエンスを選択します。
1. **削除**をクリックし、**完了**をクリックします。

## アクション

| アクション名 | AudienceStream | EventStream |
| --- | :---: | :---: |
| オーディエンスへの訪問追加 | ✓ | ✗ |
| オーディエンスからの訪問削除 | ✓ | ✗ |

**続行**をクリックしてコネクタアクションを構成します。アクションの名前を入力し、**アクションタイプ**を選択します。

次のセクションでは、各アクションのパラメータとオプションの構成方法について説明します。

### オーディエンスへの訪問追加

#### パラメータ

|**パラメータ**| **説明**|
|---| ---|
|オーディエンスID| オーディエンスを選択またはオーディエンスIDを入力します。|
|IDFA (MD5ハッシュ適用)| プレーンテキストのiOS IDFAを提供します。コネクタはこの値を空白トリムし、小文字に変換し、MD5ハッシュを使用してハッシュ化します。|
|IDFA (MD5ハッシュ済み)| すでにMD5ハッシュされたiOS IDFAを提供します。|
|AAID (MD5ハッシュ適用)| プレーンテキストのAndroidデバイス用広告識別子を提供します。コネクタはこの値を空白トリムし、小文字に変換し、MD5ハッシュを使用してハッシュ化します。|
|AAID (MD5ハッシュ済み)| すでにMD5ハッシュされたAndroidデバイス用広告識別子を提供します。|
|IDFA (SHA256ハッシュ適用)| プレーンテキストのiOS IDFAを提供します。コネクタはこの値を空白トリムし、小文字に変換し、SHA256ハッシュを使用してハッシュ化します。|
|IDFA (SHA256ハッシュ済み)| すでにSHA256ハッシュされたiOS IDFAを提供します。|
|AAID (SHA256ハッシュ適用)| プレーンテキストのAndroidデバイス用広告識別子を提供します。コネクタはこの値を空白トリムし、小文字に変換し、SHA256ハッシュを使用してハッシュ化します。|
|AAID (SHA256ハッシュ済み)| すでにSHA256ハッシュされたAndroidデバイス用広告識別子を提供します。|
|メールアドレス (SHA256ハッシュ適用)| プレーンテキストのメールアドレス（`@`を含む必要があります）を提供します。コネクタはこの値を空白トリムし、小文字に変換し、SHA256ハッシュを使用してハッシュ化します。 |
|メールアドレス (SHA256ハッシュ済み)| すでにSHA256ハッシュされたメールアドレスを提供します。 |
|電話番号 (SHA256ハッシュ適用)| プレーンテキストの電話番号を提供します。コネクタはこの値を空白トリムし、SHA256ハッシュを使用してハッシュ化します。|
|電話番号 (SHA256ハッシュ済み)| すでにSHA256ハッシュされた電話番号を提供します。|
