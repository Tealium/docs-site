イプシロンはマーケティングサービスプロバイダーです。イプシロンイベントAPIを使用して、お客様のウェブサイトでの閲覧や購入情報をイプシロンに伝えます。

## 構成の構成

コネクタマーケットプレイスにアクセスし、新しいコネクタを追加します。コネクタを追加する一般的な手順については、[コネクタについて]()を参照してください。

コネクタを追加した後、以下の構成を構成します：

* **会社ID**：コールを識別するための会社ID（`dtm_cid`）。
* **セカンダリ会社ID**：会社IDを確認するための別の識別子（`dtm_cmagic`）。

## アクション

| **アクション名** | **AudienceStream** | **EventStream** |
| --- | --- | --- |
| ブラウジングイベントの送信 | ✗ | ✓ |
| トランザクションイベントの送信 | ✗ | ✓ |

アクションの名前を入力し、ドロップダウンメニューからアクションタイプを選択します。

次のセクションでは、各アクションのパラメータとオプションの構成方法について説明します。

### ブラウジングイベントの送信

#### イベントデータ

| **パラメータ** | **説明** |
| --- | --- |
| フォームID（`dtm_fid`） | （必須）ブラウジングイベントのフォームID。 |
| URL（`dtmc_loc`） | （必須）ユーザーが閲覧しているページのURL。 |
| IPアドレス（`dtm_user_ip`） | （必須）ユーザーのIPアドレス。 |
| Cookie ID（`dtm_cookie_id`） | （必須）ユーザーのユニークID。 |
| プロモID（`dtm_promo_id`） | ページタイプまたはプロモーションコンテキストを識別する値。 |
| ブランド（`dtmc_brand`） | 閲覧された製品またはコンテンツに関連するブランド。 |
| 正規URL（`dtmc_canonical_url`） | 閲覧中のページの正規URL。 |
| カテゴリ（`dtmc_category`） | 主要な製品またはコンテンツのカテゴリ。 |
| 顧客ID（`dtm_user_id`） | 既知の顧客のユニーク識別子（例：CRM IDやロイヤリティID）。 |
| 部門（`dtmc_department`） | 製品に関連する部門または事業部（例：`Electronics`, `Home`）。 |
| メール（SHA256ハッシュ適用） | SHA256ハッシュを適用する前のプレーンテキストのメールアドレス。 |
| メール（既にSHA256ハッシュ済み） | SHA256でハッシュされたメールアドレス；提供された通りにイプシロンに送信。 |
| GDPR同意（`dtmc_tcf_string`） | 訪問のGDPR同意状態を表すIAB TCF同意文字列。 |
| メーカーモデル部品番号（`dtmc_mpn`） | 製品のメーカーモデルまたは部品番号。 |
| メーカーUPC（`dtmc_upc`） | 製品のメーカーUPC。 |
| 製品ID（`dtmc_product_id`） | 閲覧されている製品のユニーク識別子。 |
| 参照URL（`dtm_referrer`） | 訪問を現在のページに誘導したページのURL。 |
| サブカテゴリ（`dtmc_sub_category`） | 製品またはコンテンツのサブカテゴリ（例：`Shoes` within `Apparel`）。 |
| ユーザーエージェント（`dtm_user_agent`） | 訪問のデバイスからのブラウザユーザーエージェント文字列。 |
| キャッシュバスター | 応答キャッシングを防ぐために使用されるランダムまたはタイムスタンプ値。 |
| GPP文字列 | （必須）グローバルプライバシープラットフォーム（GPP）同意文字列。 |
| カスタムクエリパラメータ | 基本URLにクエリ文字列パラメータとして追加するカスタム値。 |

| IPアドレス（`dtm_user_ip`） | （必須）ユーザーのIPアドレス。 |
| Cookie ID（`dtm_cookie_id`） | （必須）ユーザーを識別するために広告主が提供したCookie ID。 |
| プロモID（`dtm_promo_id`） | トランザクションに関連するページタイプまたはプロモーションを識別する値。 |
| ブランド（`dtmc_brand`） | 購入された製品に関連するブランド。 |
| 正規URL（`dtmc_canonical_url`） | トランザクションページの正規URL。 |
| カテゴリ（`dtmc_category`） | 主要な製品またはコンテンツのカテゴリ。 |
| 顧客ID（`dtm_user_id`） | 既知の顧客のユニーク識別子（例：CRM IDやロイヤリティID）。 |
| 部門（`dtmc_department`） | 製品に関連する部門または事業部。 |

| 製品ID（`dtmc_product_id`） | 購入された製品のユニーク識別子。 |
| 参照URL（`dtm_referrer`） | 訪問をトランザクションページに誘導したページのURL。 |
| サブカテゴリ（`dtmc_sub_category`） | 製品またはコンテンツのサブカテゴリ。 |
| ユーザーエージェント（`dtm_user_agent`） | 訪問のデバイスからのブラウザユーザーエージェント文字列。 |

#### トランザクションデータ

| **パラメータ** | **説明** |
| --- | --- |
| トランザクションID（`dtmc_transaction_id`） | 広告主の各コンバージョンに対するユニーク識別子。 |
| 注文小計（`dtm_conv_val`） | 購入されたすべてのアイテムの合計価格。税金や送料は含まないでください。 |
| コンバージョンタイプ（`dtmc_conv_type`） | 配送やピックアップなど、オンライン購入のタイプを区別するために使用。 |
| 通貨コード（`dtm_conv_curr`） | ISO 4217通貨コード（例：EURやUSD）。 |
| 店舗の場所（`dtmc_conv_store_location`） | ピックアップタイプのコンバージョンの店舗の場所。 |

#### アイテムデータ

| **パラメータ** | **説明** |
| --- | --- |
| アイテムID | トランザクション内のアイテムIDの配列。 |
| アイテム価格 | トランザクション内のアイテム価格の配列。 |
| アイテム数量 | トランザクション内のアイテム数量の配列。 |
| アイテム割引 | トランザクション内のアイテム割引の配列。 |
| カスタムクエリパラメータ | 基本URLにクエリ文字列パラメータとして追加するカスタム値。
