## コネクタのアクション

| アクション名 | AudienceStream | EventStream |
| --- | :---: | :---: |
|リストにユーザーを追加| ✓| ✗|
|リストからユーザーを削除| ✓| ✗|
|リストにユーザーを追加（バッチジョブ）| ✓| ✗|
|リストからユーザーを削除（バッチジョブ）| ✓| ✗|
|トランザクションメールを送信| ✓| ✗|

## 構成の構成

コネクタマーケットプレイスに移動し、新しいコネクタを追加します。コネクタの追加方法の一般的な指示については、[Connector Overview](https://docs.tealium.com/about-connectors/)の記事を参照してください。

コネクタを追加した後、以下の構成を構成します：

* **APIキー**
  * Sailthruアカウント機能にアクセスするためのAPIキー。

* **共有シークレット**
  * Sailthruアカウントの共有シークレット。

### アクション - リストにユーザーを追加（バッチジョブ）

##### バッチ制限

このアクションは、ベンダーへの大量データ転送をサポートするためにバッチリクエストを使用します。詳細については、[Batched Actions](https://docs.tealium.com/batched-actions/)を参照してください。リクエストは、以下のいずれかの閾値が達成されるか、プロファイルが公開されるまでキューに入れられます：

* リクエストの最大数：10000
* 最古のリクエストからの最大時間：10分
* リクエストの最大サイズ：10MB

| **パラメータ**             | **説明**                                                                     |
| ------------------------- | ----------------------------------------------------------------------------------- |
| Sailthruリスト             | <ul><li>ユーザーを追加するリストを1つ以上選択します。</li></ul>                          |
| ユーザーFacebook ID          | <ul><li>ユーザーFacebook IDにマッピング</li></ul>                                           |
| Sailthru Horizonクッキー   | <ul><li>Sailthru Horizonクッキーにマッピング</li></ul>                                    |
| ユーザーSMS                  | <ul><li>ユーザーSMSにマッピング</li></ul>                                                   |
| ユーザー指定識別子 | <ul><li>ユーザー指定識別子にマッピング</li></ul>                                  |
| ユーザーEmail                | <ul><li>ユーザーEmailにマッピング</li></ul>                                                 |
| ユーザーTwitter名         | <ul><li>ユーザーX名にマッピング</li></ul>                                          |
| Sailthru識別子       | <ul><li>Sailthru識別子にマッピング</li></ul>                                        |
| ユーザーカスタム変数     | <ul><li>ユーザーにカスタム変数を構成します。</li></ul>                                 |
| Opt Out                   | <ul><li>次のいずれかでなければなりません：`all`, `basic`, `blast`または`none`</li></ul> |
| サインアップ日               | <ul><li>ユーザーのサインアップ日。</li></ul>                                                 |
