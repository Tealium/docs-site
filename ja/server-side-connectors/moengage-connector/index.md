## 構成の構成

コネクタマーケットプレイスに移動し、新しいコネクタを追加します。コネクタの追加方法の一般的な指示については、[コネクタ概要]()の記事を参照してください。

コネクタを追加したら、次の構成を構成します：

* **ユーザー名**
(必須): MoEngageアカウントのユーザー名は、MoEngageダッシュボードで利用可能です。詳細については、[Moengageドキュメンテーション](https://developers.moengage.com/hc/en-us/articles/4404674776724-Overview#01H815VFPB3Y06MX539856QSK4)を参照してください。

* **パスワード**
(必須): MoEngageアカウントのパスワードは、MoEngageダッシュボードで利用可能です。詳細については、[Moengageドキュメンテーション](https://developers.moengage.com/hc/en-us/articles/4404674776724-Overview#01H815VFPB3Y06MX539856QSK4)を参照してください。

* **データセンター**
(必須): あなたのAPIが使用するデータセンター。詳細については、[Moengageデータセンタードキュメンテーション](https://help.moengage.com/hc/en-us/articles/360057030512-Data-Centers-in-MoEngage#01G5DQVXGT2KZMXTJPF77QPJ25)を参照してください。期待値：0-4。

## アクション

アクションの名前を入力し、ドロップダウンメニューからアクションタイプを選択します。

このセクションでは、各アクションのパラメータとオプションの構成方法について説明します。

### ユーザー情報の追加/変更 (リアルタイム)

#### パラメータ

| **パラメータ** | **説明** |
| --- | --- |
| 顧客ID | (必須) MoEngage SDKから構成および渡される一意の識別子。 |
| 名前 | ユーザーのフルネーム。 |
| 名 | ユーザーの名前。 |
| 姓 | ユーザーの姓。 |
| メール | ユーザーのメールアドレス。 |
| 年齢 | ユーザーの年齢。 |
| 性別 | ユーザーの性別。 |
| モバイル番号 | ユーザーの携帯電話番号。例：**918888444411**。 |
| 緯度 | ユーザーの位置の緯度。 |
| 経度 | ユーザーの位置の経度。 |
| パブリッシャー名 | インストールのパブリッシャー名。例：**Google Ads**。 |
| 初回訪問 | ユーザーが作成された時間（ISO 8601形式）。例：**2019-05-21T03:47:35Z**。 |
| 最終訪問 | ユーザーの最終訪問時間（ISO 8601形式）。例：**2020-05-01T03:52:35Z**。 |

| LTV | ユーザーのライフタイムバリュー。 |

| プラットフォーム | 値はユーザーのアクティブなプラットフォームに基づいています。許可される値は**ANDROID**、**iOS**、または**web**です。 |
| アクティブ | プラットフォームのステータス。 |

### ユーザー情報の追加/変更 (バッチ処理)

#### バッチ制限

このアクションは、ベンダーへの大量のデータ転送をサポートするためにバッチリクエストを使用します。リクエストは、以下のいずれかの閾値が満たされるまでキューに入れられます：

* リクエストの最大数：50
* 最も古いリクエストからの最大時間：10分
* リクエストの最大サイズ：1 MB

### イベントの追跡 (リアルタイム)

#### パラメータ

| **パラメータ** | **説明** |
| --- | --- |
| 顧客ID | (必須) MoEngage SDKから構成および渡される一意の識別子。 |
| アクション | (必須)  |
| デバイスID | デフォルト値は**顧客ID**の値です。この値は特定のデバイスにイベントをマッピングするために使用されます。 |
| プラットフォーム | イベントが発生したプラットフォームを識別するために使用されます。許可される値は**ANDROID**、**iOS**、**web**、または**unknown**です。 |
| アプリバージョン | イベントが発生したアプリのバージョン。 |
| ローカル時間 | イベントが発生したローカル時間（秒）。 |
| UTC時間 | イベントが発生したUTC時間（秒）。 |
| ユーザータイムゾーンオフセット | -54000から54000までの秒数で値を持つべきです。例えば、IST（UTC&#43;0530）の場合、**ユーザータイムゾーンオフセット**は19800になります。 |

### ユーザーをオーディエンス/コホートに追加

#### パラメータ

| **パラメータ** | **説明** |
| --- | --- |
| ユーザーID | 登録済み/ログイン済みユーザーのユーザーID。**ユーザーID**または**匿名ID**のいずれかを提供する必要があります。両方が存在する場合、まず**ユーザーID**が使用されてユーザーが解決され、ユーザーが利用できない場合は**匿名ID**に基づいてユーザーが解決されます。 |
| 匿名ID | 未登録/非ログインユーザーの識別子。 |
| タイムスタンプ | リクエストのタイムスタンプ（ISO 8601）。例：**2019-05-21T03:47:35Z**。マッピングされていない場合、現在のタイムスタンプで初期化されます。 |
| コホート名 | コホートの名前。この構成は、**Tealium Audience**セクションで選択されたTealium Audience名を上書きします。 |
| コホートID | コホートのID。この構成は、**Tealium Audience**セクションで選択されたTealium Audience IDを上書きします。 |
| コホートの説明 | コホートの説明。 |
| Tealium Audience | Tealium Audienceを選択して、Tealium Audience IDとTealium Audience名をMoEngageコホートのIDと名前として使用します。&lt;br&gt; |
