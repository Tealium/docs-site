## API 情報

このコネクタは以下のベンダー API を使用します：

* API 名: ActiveCampaign API
* API バージョン: v3
* ドキュメント: [ActiveCampaign API](https://developers.activecampaign.com/reference/overview)

## 構成

コネクタマーケットプレイスにアクセスして新しいコネクタを追加します。コネクタを追加する一般的な手順については、[コネクタについて](https://docs.tealium.com/about-connectors/)を参照してください。

コネクタを追加した後、以下の構成を構成します：
* **アカウント名**  
  * アカウントに固有のベース URL を使用して API にアクセスします。
  * ActiveCampaign で **構成 &gt; 開発者** をクリックして、API URL を取得します。
* **API トークン**  
  * ActiveCampaign で **構成 &gt; 開発者** をクリックして、API トークンを取得します。
  * ActiveCampaign アカウントの各ユーザーには独自の API トークンがあります。

## アクション

| アクション名 | AudienceStream | EventStream |
| --- | :---: | :---: |
| コンタクトのアップサート | ✓ | ✓ |
| コンタクトにタグを追加 | ✓ | ✓ |
| コンタクトをオートメーションに追加 | ✓ | ✓ |
| コンタクトのリストステータスを更新 |  ✓ | ✓ |

アクションの名前を入力し、ドロップダウンメニューからアクションタイプを選択します。

次のセクションでは、各アクションのパラメータとオプションの構成方法について説明します。

### アクション - コンタクトのアップサート

#### コンタクトパラメータ

| パラメータ | 説明 |
| --- | --- |
| メール | (必須) 同期するコンタクトのメールアドレス。 |
| 名 | (必須) コンタクトの名。 |
| 姓 | (必須) コンタクトの姓。 |
| 電話番号 | (必須) コンタクトの電話番号。 |

#### フィールド値

コンタクトのフィールド値の配列。顧客のフィールド値は `id` で参照され、フィールド名ではありません。これは整数値であり、ActiveCampaign のフィールド管理を通じて見つける必要があります。

| パラメータ | 説明 |
| --- | --- |
| フィールド | フィールド名。 |
| 値 | フィールドの値。 |

### アクション - コンタクトにタグを追加

#### パラメータ

| パラメータ | 説明 |
| --- | --- |
| コンタクトID | コンタクトのID。 |
| タグID | 訪問に割り当てるタグの正確な名前を入力します。ドロップダウンからタグを選択します。 |

### アクション - コンタクトをオートメーションに追加

#### パラメータ

| パラメータ | 説明 |
| --- | --- |
| コンタクトID | コンタクトのID。 |
| オートメーションID | オートメーションのID。 |

### アクション - コンタクトのリストステータスを更新

#### パラメータ

| パラメータ | 説明 |
| --- | --- |
| リストID | (必須) コンタクトを登録するリストのID。 |
| コンタクトID | <ul><li>**コンタクトID** または **メール** のいずれかが必要です。リストに登録するコンタクトのID。</li></ul> |
| メールアドレス |  <ul><li>**コンタクトID** または **メール** のいずれかが必要です。</li><li>コンタクトのメール。</li><li>このフィールドは、ユーザーが **コンタクトID** を Tealium に保存していない場合に必要です。**コンタクトID** を使用しない場合は、API コールを通じてユーザーのメールアドレスを検索し、応答の **ID** フィールドを **コンタクトID** として使用します。</li></ul> |
| ステータス | <ul><li>(必須) `1` に構成してコンタクトをリストに登録します。</li><li>`2` に構成してコンタクトをリストから登録解除します。</li><li>警告: 登録解除からアクティブにステータスを変更すると、手動で登録解除したリストにコンタクトを再登録することができます。</li></ul> |
| ソースID | <ul><li>(オプション) コンタクトの起源に関連する一意の識別子。</li><li>リストにコンタクトを再登録する場合は `4` に構成します。</li></ul> |
