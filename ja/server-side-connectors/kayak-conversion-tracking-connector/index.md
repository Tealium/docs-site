## 構成の構成

コネクタマーケットプレイスに移動し、新しいコネクタを追加します。コネクタの追加方法については、[コネクタの概要]()の記事を参照してください。

コネクタを追加した後、以下の構成を構成します：

## アクションの構成 - パラメータとオプション

**次へ**をクリックするか、**アクション**タブに移動します。ここでコネクタのアクションを構成します。

このセクションでは、各アクションのパラメータとオプションの構成方法について説明します。

### アクション - コンバージョン送信

#### パラメータ

| **パラメータ**          | **説明**                                                                                                                                                                                                                                                                                                                                                                                                                             |
|:-----------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| KAYAKクリックID         | &lt;ul&gt;&lt;li&gt;必須&lt;/li&gt;&lt;li&gt;サイトに送信されたKAYAKのクリック/リードの固有識別子です。&lt;/li&gt;&lt;li&gt;この値は大文字と小文字を区別します。&lt;/li&gt;&lt;/ul&gt;                                                                                                                                                                                                                                                                                                  |
| 予約ID             | &lt;ul&gt;&lt;li&gt;必須&lt;/li&gt;&lt;li&gt;予約の固有識別子です。&lt;/li&gt;&lt;li&gt;これは予約の固有識別子であり、すべての後続のレポートで使用されます。&lt;/li&gt;&lt;/ul&gt;                                                                                                                                                                                                                                                                  |
| KAYAKコミッション       | &lt;ul&gt;&lt;li&gt;必須&lt;/li&gt;&lt;li&gt;予約に対するKAYAKの収益で、コミッション通貨パラメータで指定された通貨です。&lt;/li&gt;&lt;li&gt;通貨記号（$など）は含めないでください。&lt;/li&gt;&lt;li&gt;小数点以下は6桁に制限してください。&lt;/li&gt;&lt;/ul&gt;                                                                                                                                                                                                           |
| コミッション通貨    | &lt;ul&gt;&lt;li&gt;必須&lt;/li&gt;&lt;li&gt;Kayakコミッションの通貨です。&lt;/li&gt;&lt;li&gt;可能な場合は、KAYAKへのコミッション支払いに使用される通貨を指定してください。&lt;/li&gt;&lt;li&gt;[ISO 4217](https://en.wikipedia.org/wiki/ISO_4217)コードを使用してください。&lt;/li&gt;&lt;/ul&gt;                                                                                                                                                                                                           |
| コンバージョン時刻        | &lt;ul&gt;&lt;li&gt;オプション&lt;/li&gt;&lt;li&gt;予約が最初に行われた日時です。レポートのタイムゾーンで指定してください。&lt;/li&gt;&lt;li&gt;`YYYY-MM-DDThh:mm:ssTZD`形式を使用し、[W3CによるISO 8601の簡略化](https://www.w3.org/TR/NOTE-datetime)を使用してください。&lt;/li&gt;&lt;li&gt;指定されていない場合、現在のタイムスタンプが使用されます。&lt;/li&gt;&lt;/ul&gt;                                                                                                                                     |
| 価格                  | &lt;ul&gt;&lt;li&gt;予約の総額です。通貨パラメータで指定された通貨で表記してください。&lt;/li&gt;&lt;li&gt;通貨記号（$など）は含めないでください。&lt;/li&gt;&lt;li&gt;小数点以下は6桁に制限してください。&lt;/li&gt;&lt;/ul&gt;                                                                                                                                                                                                                                            |
| 通貨               | &lt;ul&gt;&lt;li&gt;価格の通貨です。&lt;/li&gt;&lt;li&gt;[ISO 4217](https://en.wikipedia.org/wiki/ISO_4217)コードを使用してください。&lt;/li&gt;&lt;li&gt;価格パラメータを使用する場合は、このパラメータが必要です。&lt;/li&gt;&lt;/ul&gt;                                                                                                                                                                                                                                                         |
| 予約タイプ           | &lt;ul&gt;&lt;li&gt;予約のタイプです。&lt;/li&gt;&lt;li&gt;`flight`、`hotel`、`car`、`package`、`cruise`、`train`、`restaurant`、`activity`のいずれかである必要があります。&lt;/li&gt;&lt;/ul&gt;                                                                                                                                                                                                                                                                                              |
| コミッション率  | &lt;ul&gt;&lt;li&gt;この予約に適用されるコミッション率です。&lt;/li&gt;&lt;li&gt;0から1までの小数で、小数点以下は6桁に制限してください。&lt;/li&gt;&lt;/ul&gt;                                                                                                                                                                                                                                                          |
| タグ1                  | &lt;ul&gt;&lt;li&gt;追加データのためのカスタムフィールドです。&lt;/li&gt;&lt;/ul&gt;                                                                                                                                                                                                                                                                                                                                                                                         |
| タグ2                  | &lt;ul&gt;&lt;li&gt;追加データのためのカスタムフィールドです。&lt;/li&gt;&lt;/ul&gt;                                                                                                                                                                                                                                                                                                                                                                                         |
| チェックイン日          | &lt;ul&gt;&lt;li&gt;予約のチェックイン日です。&lt;/li&gt;&lt;li&gt;`yyyy-mm-dd`形式を使用してください。&lt;/li&gt;&lt;li&gt;プロパティの現地のタイムゾーンを使用してください。&lt;/li&gt;&lt;/ul&gt;                                                                                                                                                                                                                                                                                                  |
| チェックアウト日         | &lt;ul&gt;&lt;li&gt;予約のチェックアウト日です。&lt;/li&gt;&lt;li&gt;`yyyy-mm-dd`形式を使用してください。&lt;/li&gt;&lt;li&gt;プロパティの現地のタイムゾーンを使用してください。&lt;/li&gt;&lt;/ul&gt;                                                                                                                                                                                                                                                                                                       |
| HID                    | &lt;ul&gt;&lt;li&gt;予約されたプロパティの固有識別子です。これは、データフィードで提供され、APIで使用されるものと同じです。&lt;/li&gt;&lt;/ul&gt;                                                                                                                                                                                                                                                                                                 |
| 旅行者の数    | &lt;ul&gt;&lt;li&gt;旅行者の総数（全年齢）です。&lt;/li&gt;&lt;/ul&gt;                                                                                                                                                                                                                                                                                                                                                                                     |
| 部屋の数        | &lt;ul&gt;&lt;li&gt;予約された部屋の数です。&lt;/li&gt;&lt;/ul&gt;                                                                                                                                                                                                                                                                                                                                                                                               |
| フライト出発地          | &lt;ul&gt;&lt;li&gt;予約されたフライトの初期出発空港の[IATAコード](https://en.wikipedia.org/wiki/IATA_airport_code)です。&lt;/li&gt;&lt;/ul&gt;                                                                                                                                                                                                                                                                                                      |
| フライト目的地     | &lt;ul&gt;&lt;li&gt;予約されたフライトの最初のレグの目的地空港の[IATAコード](https://en.wikipedia.org/wiki/IATA_airport_code)です。&lt;/li&gt;&lt;/ul&gt;                                                                                                                                                                                                                                                                                        |
| 運賃クラス             | &lt;ul&gt;&lt;li&gt;各セグメントの運賃基準コードのカンマ区切りのリストです。&lt;/li&gt;&lt;li&gt;出発日で順序付ける必要があります。&lt;/li&gt;&lt;li&gt;すべてのフライトセグメントが同じ運賃基準コードを共有する場合は、単一の値を使用できます。&lt;/li&gt;&lt;li&gt;複数の運賃クラスを追加するには、配列タイプの属性をマップします。&lt;/li&gt;&lt;li&gt;単一の値の属性を追加するには、単一の運賃クラスをマップします。&lt;/li&gt;&lt;/ul&gt;                                                                                |
| キャビンタイプ             | &lt;ul&gt;&lt;li&gt;各セグメントの予約されたキャビンクラスのカンマ区切りのリストです。&lt;/li&gt;&lt;li&gt;可能な値は`First`、`Business`、`Premium Economy`、`Economy`です。&lt;/li&gt;&lt;li&gt;出発日で順序付ける必要があります。&lt;/li&gt;&lt;li&gt;すべてのフライトセグメントが同じキャビンクラスを共有する場合は、単一の値を使用できます。&lt;/li&gt;&lt;li&gt;複数のキャビンタイプを追加するには、配列タイプの属性をマップします。&lt;/li&gt;&lt;li&gt;単一の値の属性を追加するには、単一のキャビンタイプをマップします。&lt;/li&gt;&lt;/ul&gt; |
| レグインデックス              | &lt;ul&gt;&lt;li&gt;0から始まるレグ番号です。&lt;/li&gt;&lt;/ul&gt;                                                                                                                                                                                                                                                                                                                                                                                                  |
| セグメントインデックス          | &lt;ul&gt;&lt;li&gt;0から始まるセグメント番号です。&lt;/li&gt;&lt;/ul&gt;                                                                                                                                                                                                                                                                                                                                                                                              |
| 出発空港コード | &lt;ul&gt;&lt;li&gt;出発空港コード&lt;/li&gt;&lt;/ul&gt;                                                                                                                                                                                                                                                                                                                                                                                                    |
| 到着空港コード   | &lt;ul&gt;&lt;li&gt;到着空港コード&lt;/li&gt;&lt;/ul&gt;                                                                                                                                                                                                                                                                                                                                                                                                      |
| 出発日時    | &lt;ul&gt;&lt;li&gt;`yyyy-mm-dd`形式を使用してください。&lt;/li&gt;&lt;/ul&gt;                                                                                                                                                                                                                                                                                                                                                                                              |
| 到着日時      | &lt;ul&gt;&lt;li&gt;`yyyy-mm-dd`形式を使用してください。&lt;/li&gt;&lt;/ul&gt;                                                                                                                                                                                                                                                                                                                                                                                              |
| キャリアコード           | &lt;ul&gt;&lt;li&gt;[IATAコード](https://en.wikipedia.org/wiki/IATA_airport_code)&lt;/li&gt;&lt;/ul&gt;                                                                                                                                                                                                                                                                                                                                                              |
| フライト番号          | &lt;ul&gt;&lt;li&gt;フライト番号です。&lt;/li&gt;&lt;li&gt;複数のフライトの予定を追加するには、配列タイプの属性をマップします。&lt;/li&gt;&lt;li&gt;配列タイプの属性は同じ長さである必要があります。&lt;/li&gt;&lt;li&gt;単一の値の属性を追加するには、単一のフライト予定をマップします。&lt;/li&gt;&lt;/ul&gt;                                                                                                                                                                                                         |
| レンタカーのピックアップ日       | &lt;ul&gt;&lt;li&gt;レンタカーのピックアップ日です。ピックアップ場所の現地のタイムゾーンを使用してください。&lt;/li&gt;&lt;li&gt;`yyyy-mm-dd`形式を使用してください。&lt;/li&gt;&lt;/ul&gt;                                                                                                                                                                                                                                                                                                             |
| レンタカーの返却日      | &lt;ul&gt;&lt;li&gt;レンタカーの返却日です。返却場所の現地のタイムゾーンを使用してください。&lt;/li&gt;&lt;li&gt;`yyyy-mm-dd`形式を使用してください。&lt;/li&gt;&lt;/ul&gt;                                                                                                                                                                                                                                                                                                          |
| レンタカーのピックアップID         | &lt;ul&gt;&lt;li&gt;この予約のピックアップ場所の固有識別子です。これは、データフィードで提供され、APIで使用されるものと同じです。&lt;/li&gt;&lt;/ul&gt;                                                                                                                                                                                                                                                                                                 |
| レンタカーの返却ID        | &lt;ul&gt;&lt;li&gt;この予約の返却場所の固有識別子です。これは、データフィードで提供され、APIで使用されるものと同じです。&lt;/li&gt;&lt;/ul&gt;                                                                                                                                                                                                                                                                              |
| レンタカー会社             | &lt;ul&gt;&lt;li&gt;この予約のレンタカー会社の固有識別子です。これは、データフィードで提供され、APIで使用されるものと同じです。&lt;/li&gt;&lt;/ul&gt;                                                                                                                                                                                                                                                                                     |
