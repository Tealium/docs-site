## API情報

このコネクタは以下のベンダーAPIを使用します：

* API名：Cordial API
* APIバージョン：v1.0
* ドキュメント：[Cordial API](https://support.cordial.com/hc/en-us/categories/12092749097741-Developer-resources)

## 構成

コネクタマーケットプレイスに移動して、新しいコネクタを追加します。コネクタを追加する一般的な手順については、[コネクタについて](https://docs.tealium.com/about-connectors/)を参照してください。

コネクタを追加した後、以下の構成を構成します：

* **APIキー**  
CordialはHTTP Basic認証（BA）を使用します。Cordialプラットフォーム内で、アカウント用のエンコードされたAPIキーを生成できます。このキーはBA認証の`username`として使用されます。詳細については、[RESTful APIの概要と使用法](https://support.cordial.com/hc/en-us/articles/203885498-RESTful-API-summary-and-usage)を参照してください。
<blockquote>
Cordialで`API_KEY`ホワイトリストに[Tealium IPアドレス]()を追加する必要があります。
</blockquote>

* **Path-URL**  
以下のPath URLから選択するか、手動で入力します。
  * `https://api.cordial.io/v2/` - 管理パネルのURLが`https://admin.cordial.io/`の場合、このURLを使用します。
  * `https://api.usw2.cordial.io/v2/` - 管理パネルのURLが`https://usw2.admin.cordial.io/`の場合、このURLを使用します。

コネクタの構成が完了したら、**完了**をクリックします。

**続行**をクリックして、コネクタアクションを構成します。アクションの名前を入力し、ドロップダウンメニューからアクションタイプを選択します。

次のセクションでは、各アクションのパラメータとオプションの構成方法について説明します。

### コンタクトのアップサート

#### パラメータ

| **パラメータ** | **説明** |
| --- | --- |
| コンタクト識別キー | コンタクト識別キー。例：`email`, `cID`。 |
| コンタクト識別値 | コンタクト識別値。 |
| メールアドレス | コンタクトを識別および参照するためのユニークなチャネルアドレス。 |
| 購読ステータス | `subscribed`に構成されていない限り、プロモーションメッセージは送信されません。デフォルトは`none`。可能な値：`none`, `subscribed`, `unsubscribed`。 |
| トリガーの抑制 | `true`の場合、コンタクト属性値の更新に基づくメッセージトリガーを抑制します。デフォルトは`true`。 |
| 強制購読 | 購読解除されている場合、このキーを使用して購読ステータスを購読に更新する必要があります。可能な値：`true`, `false`。 |
| 無効 | `false`に構成されていない限り、メールメッセージは送信されません。デフォルトは`false`。可能な値：`true`, `false`。 |

#### オプショナルパラメータ

| **パラメータ** | **説明** |
| --- | --- |
| アカウントリスト | リストからコンタクトを追加または削除します。可能な値：true, false。 |
| 文字列パラメータ | 属性タイプが文字列として構成されている場合、文字列属性値を追加します。 |
| 数値パラメータ | 属性タイプが数値として構成されている場合、数値属性値を追加します。 |
| 日付パラメータ | 属性タイプが日付として構成されている場合、日付属性値を追加します。 |
| ジオパラメータ | 属性タイプがジオとして構成されている場合、ジオ属性値を追加します。ジオ属性には次のキー名のスキーマが含まれています：<ul><li>住所</li><li>住所2</li><li>市区町村</li><li>州</li><li>郵便番号</li><li>国ISO</li><li>タイムゾーン</li><li>位置</li><ul><li>緯度</li><li>経度</li></ul></ul> |
| 配列パラメータ | 属性タイプが配列として構成されている場合、配列属性値を追加します。デフォルトの動作は、提供された値で配列値を置き換えることです。デフォルトをオーバーライドするために、追加/削除の動作を指定するオブジェクトを指定することができます。最大アイテム数の制限（デフォルトは25）を超える場合、新しい値は最も古い値を置き換えます。 |

### コンタクト活動の記録（イベント）

#### パラメータ

| **パラメータ** | **説明** |
| --- | --- |
| コンタクト識別キー | コンタクト識別キー。例：`email`, `cID`。 |
| コンタクト識別値 | コンタクト識別値。 |
| アクション | カスタムアクションを定義します。例えば、閲覧、注文、カートなどのカスタムアクション。パラメータの最大長は`40`文字です。`POST`メソッドを通じて追加できない予約済みアクションに注意してください。詳細については、[コンタクト活動（イベント）API](https://support.cordial.com/hc/en-us/articles/203885968-Contact-activities-events-API#reservedEvents)を参照してください。 |
| アクションタイムスタンプ | アクションタイムスタンプ。未定義の場合、現在の日付と時刻が使用されます。日付形式は`ISO 8601`標準です。 |
| プロパティ | 追加のイベント属性のオブジェクト。イベントが持つことができるプロパティのデフォルトの最大数は`1,000`です。数値のみの値（例：`57`）やドットを含むキー（例：`shoes.color`）は削除されます。 |

### コンタクト活動を記録（バッチ）

#### バッチ制限

このアクションは、ベンダーへの大量データ転送をサポートするためにバッチリクエストを使用します。次のいずれかの閾値に達するまでリクエストはキューに入れられます：

* 最大リクエスト数：99
* 最古のリクエストからの最大時間：10分
* リクエストの最大サイズ：10 MB

#### パラメータ

| **パラメータ** | **説明** |
| --- | --- |
| ホスト | Cordialリージョンを指定します：<ul><li>usw1: `https://integrations-ingest-svc.cordial.com`</li><li>usw2: `https://integrations-ingest-svc.usw2.cordial.com`</li><li>use1: `https://integrations-ingest-svc.use1.cordial.com`</li></ul> |
| ソース | データソース。デフォルト値は `TEALIUM` です。 |

#### 識別子

| **パラメータ** | **説明** |
| --- | --- |
| コンタクトID | コンタクトID。 |
| メール | コンタクトのメールアドレス。 |
| 電話 | コンタクトの電話番号。 |
| カスタムID | アカウントで構成された主キーまたは副キー。詳細については、[Cordial: 主キー対副キー](https://support.cordial.com/hc/en-us/articles/13084804213517-Primary-v-secondary-keys)を参照してください。 |
| 識別子の優先順位 | コンタクトを検索する際に使用される識別子の順序。 |

#### コンタクトオプション

| **パラメータ** | **説明** |
| --- | --- |
| 強制購読 | 強制購読。 |
| マージ許可 | マージ許可。 |
| マージ可能な副キー | `email` など、コンタクトレコードをマージするために使用できる副キー。 |
