## API情報

このコネクタは以下のベンダーAPIを使用します：

* API名：Yahoo API
* APIバージョン：v1
* ドキュメント：[Yahoo: Ad Tech DataX API仕様](https://help.yahooinc.com/datax/docs/yahoo-ad-tech-datax-api-specifications)
* APIエンドポイント：`https://dataxonline.yahoo.com`

## 構成

コネクタマーケットプレイスにアクセスし、新しいコネクタを追加します。コネクタの追加方法については、[コネクタについて](https://docs.tealium.com/about-connectors/)を参照してください。

コネクタを追加した後、以下の構成を構成します：

* **クライアントID**  
  * （必須）クライアントIDはYahooから提供される暗号化された資格情報ファイルにあります。
* **クライアントシークレット**  
  * （必須）クライアントシークレットはYahooから提供される暗号化された資格情報ファイルにあります。

## アクション

| アクション名 | AudienceStream | EventStream |
| --- | :---: | :---: |
| ピクセルとイベントデータの送信 | ✓ | ✓ |

アクションの名前を入力し、ドロップダウンメニューからアクションタイプを選択します。

次のセクションでは、各アクションのパラメータとオプションの構成方法について説明します。

### ピクセルとイベントデータの送信

#### パラメータ

| **パラメータ** | **説明** |
| --- | --- |
| アクションソース | （必須）イベントのソース。例：`Website`、`App`、または`Offline`。 |
| ピクセルID | （必須）Yahoo DSPで作成された統合のピクセルID。 |
| アクションソースURL | ウェブサイトのURL。 |
| イベント時間 | イベントが発生した時間、エポックミリ秒UTC。**イベント時間**がマッピングされていない場合、現在のタイムスタンプが送信されます。 |

#### ユーザーデータ

| **パラメータ** | **説明** |
| --- | --- |
| Email（既にSHA256ハッシュ化されている） | すでに空白トリム、小文字化、SHA256ハッシュ化されたメールアドレスを提供します。 |
| Email（SHA256ハッシュを適用） | （SHA256ハッシュを適用）プレーンテキストのメールアドレスを提供し、コネクタがこの値を空白トリム、小文字化、SHA256ハッシュを使用してハッシュ化します。 |
| GPSAID | Google Play Store広告ID。 |
| IDFA | Apple広告ID。 |
| Yahoo ID | Cookie Sync識別子。 |
| VMCID | クリックまたは第三者の広告サーバーから収集されたURLマクロからのクリックIDの値。これは仕様のバージョン2でサポートされます。 |

#### カスタムデータ

| **パラメータ** | **説明** |
| --- | --- |
| gv | 任意の変換の金銭的価値。値のデフォルト通貨は米ドル（USD）です。これはDSPで動的変換値として報告されます。 |
| ec | イベントに関連するコンテンツカテゴリ。 |
| el | イベントカテゴリの下の下位レベルのラベル。 |
| ea | 広告主によって定義されたアクション、例えば`AddToCart`。 |
| 商品ID | 商品IDまたはカンマ区切りの複数のID。 |
| ユーザー定義 | クライアントが標準のイベントの代わりにカスタム名を構成できる最大10個のカスタムイベントラベル。名前の最大長は32文字、値の最大長は255文字です。 |
