## コネクタのアクション

| **アクション名**         | **AudienceStream** | **EventStream** |
|:----------------------|:-------------------|:----------------|
| 事前定義イベントの送信 | ✓                  | ✓               |
| カスタムイベントの送信 | ✓                  | ✓               |

## 必要条件

* TUNEアカウント
* メールアドレスとパスワード
* プライベート/APIキー
* パブリック/コンシューマキー

上記のキーを取得するには、TUNEアカウントにリンクされたメールアドレスからTUNEサポートにリクエストを送信し、キーを作成してもらう必要があります。

## 構成の構成

コネクタマーケットプレイスに移動し、新しいコネクタを追加します。コネクタの追加方法については、[コネクタの概要]()の記事を参照してください。

コネクタを追加した後、以下の構成を構成します：

* **プライベートAPIキー**
  * 必須
  * プライベートAPIキーを入力してください。
  * 詳細については、[プライベートAPIキーの取得](https://tune.docs.branch.io/measurement-docs/obtaining-your-private-api-key/)を参照してください。

* **パブリックコンシューマキー**
  * 必須
  * パブリックコンシューマキーを入力してください。

## アクションの構成 - パラメータとオプション

**次へ**をクリックするか、**アクション**タブに移動します。ここでは、コネクタのアクションを構成します。

このセクションでは、各アクションのパラメータとオプションの構成方法について説明します。
