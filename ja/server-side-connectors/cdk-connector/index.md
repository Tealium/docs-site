## API情報

このコネクタは以下のベンダーAPIを使用します：

* API名：CDK Global API
* APIバージョン：v1
* APIエンドポイント：`https://api.fortellis.io`
* ドキュメント：[CDK CRM Direct Post Sales Leads](https://apidocs.fortellis.io/apis/1b0047b1-feec-42e9-be1b-d3285493ac9a)

## 構成

コネクタマーケットプレイスにアクセスして新しいコネクタを追加します。コネクタの追加方法については、[コネクタについて]()を参照してください。

このコネクタを追加する際には、ベンダーのデータプラットフォームポリシーを受け入れるよう求められます。

コネクタを追加した後、以下の構成を構成します：
* **組織ID**  
  * Fortellis組織ID（エンティティID）を見つけるには、Fortellisアカウントにログインし、名前の上にマウスを移動して**組織管理**をクリックします。エンティティIDはページに表示されます。
* **サブスクリプションID**  
  * サブスクリプションIDを選択します。
* **エンドポイント**  
  * エンドポイントを選択します。デフォルトのエンドポイントは**Prod**です。

| 状態 | 車両を`new`または`used`として識別します。 |

| メーカー | 車両の製造者。 |
| モデル | 車両のモデル。 |
| VIN | 車両の車両識別番号。 |
| 在庫 | 車両のベンダーの在庫番号。 |
| トリム | トリムの説明。 |
| 単位 | オドメータの計測単位：`mi`はマイル、`km`はキロメートル。 |
| 値 | オドメータの読み取り値。 |
| ボディスタイル | 一般的なボディスタイル、例えば`SUV`, `Sedan`, `Coupe`など。 |
| トランスミッション | `A`（オートマチック）または`M`（マニュアル）。 |
| ドア数 | 車両のドア数。 |
| 価格タイプ | 価格のタイプ。受け入れられる値：`quote`, `offer`, `msrp`, `invoice`, `call`, `appraisal`, `asking`。 |

| 相対値 | `Delta`値と`RelativeTo`プロパティを使用して、請求書などの別の値に対するパーセンテージとして価格を表現することができます。受け入れられる値：`msrp`, `invoice`。 |
| 価格ソース | この値が来たソース、例えば`Kelley Blue Book`。 |

| ファイナンス金額通貨 | ISO 4217通貨コード。 |

| ファイナンス残高タイプ | 残高のタイプ。受け入れられる値：`finance`, `residual`。|

| 電話優先 | 電話が連絡方法として優先されるかどうかを示します（メールと比較して）。 |
| 電話番号 | 電話番号。 |
