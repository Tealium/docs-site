url: https://docs.tealium.com/ja/help/resource-center/

[Tealiumプラットフォーム](https://my.tealiumiq.com/)のヘッダーで**リソースセンター**を選択すると、リソースセンターパネルが開きます。このパネルを通じて、プラットフォームを離れることなく、ヘルプ、トレーニング、サポート、フィードバックリソースにアクセスできます。

![](/images/help/resource-center.png)

| モジュール | 説明 |
|---|---|
| 製品ガイド | プラットフォーム内の現在の位置に関連するドキュメントを表示します。異なる画面に移動するとドキュメントが更新されます。すべての製品ドキュメントを横断して検索するには、検索ボックスを使用します。 |
| ナレッジベース | [Tealiumサポートナレッジベース](https://support.tealiumiq.com/en/support/solutions)へのリンクで、ユーザーガイド、トラブルシューティング、実装戦略、FAQなどをカバーしています。 |
| フィードバック | Tealium Ideas Portalで製品アイデアを閲覧、投票、提出します。詳細については、[Tealiumフィードバックポリシー]()を参照してください。 |
| 製品アップデート | 新しいTealiumリリースや機能アップデートに関する製品内アナウンスを表示します。 |
| ケーススタディ &amp; イベント | 今後のウェビナー、Tealiumイベント、顧客の成功事例に関する製品内コンテンツを特集しています。 |
| トレーニング &amp; 認定 | [Tealium University](https://university.tealium.com/learn)へのリンクで、オンデマンドのコースや認定プログラムにアクセスできます。 |
| サポートデスク | [Tealiumサポートポータル](https://support.tealiumiq.com/en/support/tickets)へのリンクで、サポートチケットを提出できます。 |
