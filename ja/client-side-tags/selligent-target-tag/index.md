Selligentは、小売、旅行・エンターテイメント、メディア・公開、金融サービスなど、多岐にわたる業界のヨーロッパとアメリカの顧客を持つ国際的なデジタルマーケティングキャンペーン管理ソリューションのベンダーです。

## タグのヒント

マッピングを使用して：
* Universe ID の標準構成値を上書きします。
* オプションの構成パラメータを構成します。
* カスタムユーザー識別子を構成します。
* タグ名/値のペアを渡します。
* ポストトラッキングコールバック関数を指定します。

## タグの構成

タグマーケットプレイスにアクセスして新しいタグを追加します。タグを追加する方法の詳細については、[タグの管理]()を参照してください。

タグを追加する際には、以下の構成を構成します：

* **Universe ID**: あなたのSelligent Universe ID。Selligentによって提供されます。

## ロードルール

すべてのページでタグをロードするか、タグがロードされる条件を構成します。詳細については、[ロードルールについて]()を参照してください。

## データマッピング

マッピングは、データレイヤー変数からベンダータグの対応する宛先変数へデータを送信するプロセスです。詳細については、[データマッピングについて]()を参照してください。

利用可能なカテゴリは以下の通りです：

### 標準

| 変数 | 説明 |
|:---------|:------------|
| `universe_id` | Universe ID |
| `customIdentifier` | カスタムユーザー識別子 |
| `tag.mytag` | カスタムタグ |
| `useConfig` | 構成を使用 |
| `finishedCallback` | コールバック関数 |
| `async` | 非同期 |
| `isEvent` | イベントですか |
