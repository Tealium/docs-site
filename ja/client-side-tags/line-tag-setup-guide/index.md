LINEで適切な顧客を見つけてビジネスの成長を促進しましょう。LINE Adsを通じてキャンペーンの目標、ターゲティング、予算、測定をいつでも管理し、より多くの視聴者にリーチし、より良い結果を達成します。

## タグのヒント

* データマッピングを使用して、構成と標準パラメータを上書きします。
* 次のE-Commerce拡張値をサポートしています：
  * タイプ (`type`)
  * アイテムID (`itemIds`)
  * 価格 (`price`)
  * 通貨 (`currency`)
  * 注文ID (`orderId`)

## タグ構成

新しいタグを追加するためにタグマーケットプレイスに移動します。タグの追加方法については、[タグ概要](https://docs.tealium.com/about-tags/)の記事を参照してください。

タグを追加する際には、以下の構成を行います：

* **タグID**：必須。あなたのLINEアカウントID。あなたのLINEアカウントIDは、カンマで連結して複数の項目を構成することができます。
* **顧客タイプ**：顧客タイプ。構成すると、顧客タイプがタグのカスタムパラメータに追加されます。空の場合、デフォルト値は`lap`です。

## ロードルール

すべてのページでタグをロードするか、タグがロードされる条件を構成します。ロードルールについての詳細は、[ロードルール](https://docs.tealium.com/about-load-rules/)のドキュメンテーションを参照してください。

## データマッピング

マッピングは、データレイヤー変数からベンダータグの対応する宛先変数にデータを送信するプロセスです。変数をタグの宛先にマッピングする方法については、[データマッピング](https://docs.tealium.com/ja/iq-tag-management/data-mappings/manage/)を参照してください。

利用可能なカテゴリは以下の通りです：

### 構成パラメータ

| 変数                           | 説明       |
|:-------------------------------|:------------|
| タグID (`tagId`)               | [文字列]    |
| 顧客タイプ (`customerType`)    | [文字列]    |

### 標準パラメータ

| 変数                                               | 説明       |
|:---------------------------------------------------|:------------|
| タイプ (`type`) (`_ctype`を上書き)                 | [文字列]    |
| アイテムID (`itemIds`) (`_cprod`を上書き)          | [配列]     |
| カテゴリ (`category`)                              | [文字列]    |
| キーワード (`keywords`)                            | [文字列]    |
| 価格 (`price`) (`_csubtotal`を上書き)              | [文字列]    |
| 通貨 (`currency`) (`order_currency`を上書き)       | [文字列]    |
| 注文ID (`orderId`) (`_corder`を上書き)             | [文字列]    |
| 開始日 (`startDate`)                               | [文字列]    |
| 終了日 (`endDate`)                                 | [文字列]    |

### イベント

選択したイベントをトリガーするために必要なマッピング変数の値を入力します。**CustomEvent**を選択した場合は、**カスタムイベント名**フィールドにイベントの名前を入力します。**CustomEvent**マッピングを構成し、イベントがLINEに送信されると、LINE Ads Platformアカウントでカスタムオーディエンスを作成する際に指定した**CustomEvent**名を選択できます。

イベントをマッピングするには、[イベントマッピングの作成](https://docs.tealium.com/ja/iq-tag-management/data-mappings/manage/)を参照してください。

| 変数          | 説明           |
|:--------------|:---------------|
| ページビュー   | `PageView`     |
| カテゴリビュー | `CategoryView` |
| 商品閲覧       | `ViewProduct`  |
| 検索           | `Search`       |
| カートに追加   | `AddToCart`    |
| 購入           | `Purchase`     |
| カスタムイベント | `Custom`       |
