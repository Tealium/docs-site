Crimtanは、世界12カ国に拠点を置く完全にプログラマティックなデータ駆動型のデジタルマーケティング会社です。Crimtanは、消費者データを価値ある洞察と創造的なディスプレイ広告キャンペーンに変え、企業がブランド認知度を高め、新規顧客を見つけ、顧客関係をより良く維持するのを支援します。

## タグのヒント

*   マッピングを使用して、標準の構成値を動的に上書きします。

## タグ構成

タグマーケットプレイスに移動して新しいタグを追加します。タグの追加方法の一般的な指示については、[タグ概要]()の記事を読んでください。

タグを追加する際には、以下の構成を行います：

*   **コンテナID**
*   **サイレントモード**

## ロードルール

すべてのページでタグをロードするか、タグがロードされる条件を構成します。ロードルールについての詳細は、[ロードルール]()のドキュメンテーションを参照してください。

## データマッピング

マッピングは、データレイヤー変数からベンダータグの対応する宛先変数にデータを送信するプロセスです。タグ宛先に変数をマップする方法については、[データマッピング]()を参照してください。

利用可能なカテゴリーは以下の通りです：

### 標準

| 変数 | 説明 |
| --- | --- |
| コンテナID | (`containerId`) |
| サイレントモード | (`silentMode`) |
| 同意ステータス | (`consentStatus`) |
