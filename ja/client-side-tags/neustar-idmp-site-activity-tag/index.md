Neustar IDMPサイトアクティビティタグを使用して、ウェブサイトでのユーザーの行動、例えばコンバージョンイベントなどを追跡します。

## タグのヒント

* タグ構成で**ピクセルタイプ**を選択して、正しいスクリプトタイプをロードします。
* マッピングを使用してタグ構成を上書きし、動的に構成します。

## タグ構成

新しいタグを追加するためにタグマーケットプレイスに移動します。詳細については、[タグについて]()を参照してください。

タグを追加する際には、以下の構成を行います：

* **NeustarタグID**：あなたのNeustar IDMPタグの数値。
* **ピクセルタイプ**：画像またはiframeタグタイプを生成するかを選択します。

## ロードルール

すべてのページでタグをロードするか、タグがロードされる条件を構成します。詳細については、[ロードルールについて]()を参照してください。

## データマッピング

マッピングは、データレイヤー変数からベンダータグの対応する宛先変数へデータを送信するプロセスです。詳細については、[データマッピングについて]()を参照してください。

利用可能なカテゴリは以下の通りです：

### 構成

| 変数 | タイプ | 説明 |
|:---------|:------------|:------------|
|  `pixelId`  | 文字列 | ピクセルID |
|  `pixelType`  | 文字列 | ピクセルタイプ |

### 標準

|  変数 | タイプ | 説明 | 
|:---------|:------------|:------------|
| `che`  | 文字列 | キャッシュバスター  |
|  `rev`  | 数値 | 収益 |
|  `cur`  | 文字列 | 通貨コード | 
|  `qty`  | 数値 | 注文数 |
|  `type`  | 文字列 | タイプ |
|  `ord`  | 文字列 | 注文ID |
|  `uid`  | 文字列 | 第三者ユーザーID |
|  `dedup`  | 文字列 | 重複排除ID |
|  `ip`  | 文字列 | IPアドレス |
