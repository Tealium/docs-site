ネクストドアユニバーサルトラッキングピクセルは、ウェブサイト全体で標準化されたイベントベースのトラッキングを可能にし、ネクストドアの最新の帰属と最適化モデルをサポートします。

## タグのヒント

* 標準パラメータを上書きするためにマッピングを使用します。
* Eコマース拡張パラメータをサポートします。
* 制限されたデータ使用により、広告主はネクストドアのシステムでのデータの使用方法を制御し、さまざまな米国州のプライバシー規制へのコンプライアンスをよりよくサポートできます。広告主が制限されたデータ使用を有効にすると、州固有の条件により必要とされる場合、特定の州の人々からの個人データの使用方法をネクストドアが制限します。これによりキャンペーンのパフォーマンスが低下し、リターゲティングと測定機能が制限される可能性があります。

## タグの構成

タグマーケットプレイスにアクセスして新しいタグを追加します。詳細については、[タグについて](https://docs.tealium.com/about-tags/)を参照してください。

タグを追加する際には、以下の構成を構成します：

* **ピクセルID**: あなたのネクストドアピクセルID。
* **自動ページビュー**: ページビューのトリガーを自動的に作成します。
* **自動購入イベント**: 購入イベントのトリガーを自動的に作成します。
* **自動イベントID生成**: すべてのネクストドアトラッキングイベントに対してイベントIDを自動生成します。

## ロードルール

すべてのページでタグをロードするか、タグがロードされる条件を構成します。詳細については、[ロードルールについて](https://docs.tealium.com/about-load-rules/)を参照してください。

## データマッピング

マッピングは、データレイヤー変数からベンダータグの対応する宛先変数へデータを送信するプロセスです。詳細については、[データマッピングについて](https://docs.tealium.com/about-data-mappings/)を参照してください。

利用可能なカテゴリは以下の通りです：

### 標準

| 変数 | タイプ | 説明 |
|:---------|:-----|:------------|
| `pixelId` | 文字列 | ピクセルID |
| `autoPageView` | ブール値 | 自動ページビュー |
| `autoPurchaseEvent` | ブール値 | 自動購入イベント |
| `generate_event_id` | ブール値 | イベントIDの自動生成 |
| `event_id` | 文字列 | イベントID |
| `click_id` | 文字列 | ネクストドアクリックID |
| `nickname` | 文字列 | カスタム変換のニックネーム |

### ユーザー

| 変数 | タイプ | 説明 |
|:---------|:-----|:------------|
| `userEmail` | 文字列 | ユーザーのメール |
| `userEmailHashed` | 文字列 | ハッシュ化されたユーザーのメール |

### Eコマース

| 変数 | タイプ | 説明 |
|:---------|:-----|:------------|
| `order_id` | 文字列 | 注文ID（`_corder`を上書き） |
| `order_value` | 文字列 | 注文値（`_ctotal`を上書き） |
| `id` | 配列 | 製品IDのリスト（`_cprod`を上書き） |
| `quantity` | 配列 | 数量のリスト（`_cquan`を上書き） |
| `item_price` | 配列 | 価格のリスト（`_cprice`を上書き） |
| `content_name` | 配列 | コンテンツ名のリスト |

### ユーザー属性

| 変数 | タイプ | 説明 |
|:---------|:-----|:------------|
| `city` | 文字列 | 市 |
| `country` | 文字列 | 国 |
| `date_of_birth` | 文字列 | 生年月日 |
| `external_id` | 文字列 | 外部ID |
| `email` | 文字列 | メール |
| `first_name` | 文字列 | 名 |
| `gender` | 文字列 | 性別 |
| `last_name` | 文字列 | 姓 |
| `phone_number` | 文字列 | 電話番号 |
| `state` | 文字列 | 州 |
| `street_address` | 文字列 | 住所 |
| `zip_code` | 文字列 | 郵便番号 |

### 制限されたデータ使用

| 変数 | タイプ | 説明 |
|:---------|:-----|:------------|
| `restricted_data_usage_country` | 数値 | 使用国 |
| `restricted_data_usage_state` | 数値 | 使用州 |
