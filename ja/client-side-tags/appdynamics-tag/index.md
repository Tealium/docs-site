AppDynamicsは、ソフトウェアからインフラストラクチャ、ビジネスジャーニーまで、ビジネスパフォーマンスを理解し最適化するためのアプリケーションパフォーマンス管理（APM）プラットフォームです。

## タグのヒント

* マッピングを使用して、標準の構成値を動的に上書きします。

## タグ構成

新しいタグを追加するためにタグマーケットプレイスに移動します。詳細については、[タグについて]()を参照してください。

タグを追加する際には、以下の構成を行います：

* **App Key**: AppDynamicsアカウントからのEUMアプリキー。

## ロードルール

すべてのページでタグをロードするか、タグがロードされる条件を構成します。詳細については、[ロードルールについて]()を参照してください。

## データマッピング

マッピングは、データレイヤー変数からベンダータグの対応する宛先変数にデータを送信するプロセスです。詳細については、[データマッピングについて]()を参照してください。

利用可能なカテゴリは以下の通りです：

### タグ構成

| 変数 | タイプ | 説明 |
|:---------|:------------|:------------|
| App Key   | String | AppDynamicsアカウントからのEUMアプリキー（タグ構成を上書きします）。 |

### 標準

| 変数 | タイプ | 説明 |
|:---------|:------------|:------------|
|  `xhr_url`  | String | あなたのXHR URL |
|  `error_msg`  | String | エラーメッセージ | 
|  `error_line`  | Number | エラーが発生したソースコードの行番号。 |
|  `error_stack`  | String | エラーが発生したときのスタックトレース。 |

### イベント

イベントをマップするには、[イベントマッピングの作成](/ja/iq-tag-management/data-mappings/manage/#add-an-event-mapping)を参照してください。

| 変数 | 説明 |
|:---------|:------------|
| `VPageView`  |仮想ページビューの段階を追跡します。 |
| `Ajax`| Ajaxリクエストを追跡します。 |
| `Error` | エラーを追跡します。 |

### イベント固有のパラメータ

イベントをマップするには、[イベントマッピングの作成](/ja/iq-tag-management/data-mappings/manage/#add-an-event-mapping)を参照してください。

| 変数 | 説明 |
|:---------|:------------|
| `xhr_url` | あなたのXHR URL。  |
| `error_msg` | エラーメッセージ。 |
| `error_line` | エラーが発生したソースコードの行番号。 |
| `error_stack` | エラーが発生したときのスタックトレース。 |  |
| `VPageView` | 仮想ページビューの段階を追跡します。 |
| `Ajax` | Ajaxリクエストを追跡します。 |
| `Error`| エラーを追跡します。 |
