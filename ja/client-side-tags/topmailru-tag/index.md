## タグ構成

新しいタグを追加するためにタグマーケットプレイスに移動します。詳細については、[タグについて]()を参照してください。

タグを追加する際には、以下の構成を行います：

* **カウンターID**：（必須）あなたのTop Mail.ruカウンターID。

## ロードルール

タグをすべてのページにロードするか、タグがロードされる条件を構成します。詳細については、[ロードルールについて]()を参照してください。

## データマッピング

マッピングは、データレイヤー変数からベンダータグの対応する宛先変数にデータを送信するプロセスです。詳細については、[データマッピングについて]()を参照してください。

利用可能なカテゴリは以下の通りです：

### スタンダード

| 変数 | タイプ | 説明 |
|:---------|:------------|:------------|
|  `base_url`  | 文字列 | ベースURL |
|  `id`  | 文字列 | カウンターID |

### E-commerce

| 変数 | タイプ | 説明 |
|:---------|:------------|:------------|
|  `productid`  | 文字列 | 商品ID |
|  `value`  | 文字列 | 値 |
|  `pagetype`  | 文字列 | ページタイプ |
|  `list`  | 文字列 | リスト |
|  `userid`  | 文字列 | ユーザーID |
|  `totalvalue`  | 数値または文字列 | 合計値 |

### イベント

イベントをマッピングするには、[イベントマッピングの作成]()を参照してください。

| 変数 | タイプ | 説明 |
|:---------|:------------|:------------|
| `pageView` | 文字列 | ページビュー |
| `viewProduct` | 文字列 | 商品表示 |
| `addToCart` | 文字列 | カートに追加 |
| `purchase` | 文字列 | 購入 |
| `itemView` | 文字列 | アイテム表示 |
| `custom` | 文字列 | カスタム |

### イベント固有のパラメータ

イベントをマッピングするには、[イベントマッピングの作成]()を参照してください。

| 変数 | タイプ | 説明 |
|:---------|:------------|:------------|
| `productid` | 文字列 | 商品ID |
| `value` | 文字列 | 値 |
| `pagetype` | 文字列 | ページタイプ |
| `list` | 文字列 | リスト |
| `userid` | 文字列 | ユーザーID |
| `totalvalue` | 数値または文字列 | 合計値 |
| `custom_parameter` | 文字列 | カスタムパラメータ |
| `pageView` | 文字列 | ページビュー |
| `viewProduct` | 文字列 | 商品表示 |
| `addToCart` | 文字列 | カートに追加 |
| `purchase` | 文字列 | 購入 |
| `custom_event` | 文字列 | カスタムイベント |
