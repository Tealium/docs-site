## タグ構成

タグマーケットプレイスに移動して新しいタグを追加します。タグの追加方法の一般的な指示については、[タグ概要](https://docs.tealium.com/about-tags/)の記事を参照してください。

タグを追加する際には、以下の構成を行います：

* **タイトル**（必須）：タグインスタンスを識別するための記述的なタイトルを入力します。
* **タグタイプ**（必須）：ドロップダウンリストから、読み込むタグのタイプを選択します。
  * マスタータグ：これにより、Zanoxが広告主に提供するMasterTagコードが読み込まれます。
  * PPS：Pay-per-Saleの略です。これにより、PPSトラッキングピクセルが読み込まれます。
  * PPL：Pay-per-Leadの略です。これにより、PPLトラッキングピクセルが読み込まれます。
* **PPS/PPL ID**（オプション）：PPS/PPLタグタイプを選択した場合、このフィールドにタグ識別子を入力します。IDは、Zanoxが提供するトラッキングピクセルの末尾に表示されます。MasterTagタイプを選択した場合、このフィールドは空白のままにします。例：このURLの末尾の英数字の値、//ad.zanox.com/ppl/?2173D3211067525があなたのPPL IDです。
* **デフォルトのzx\_class**（オプション）：MasterTagタイプを選択した場合、このフィールドにクラス属性の値（'zx'プレフィックスを除く）を入力します。このフィールドを動的に入力することを希望する場合、マッピングツールボックスのdivコンテナクラスIDにマップする必要があります（[マッピングの構成](#mappings)を参照）例：48EFE82D8GE72DE1BBE2


<blockquote>
Zanoxは、ページ上にMasterTagのインスタンスを1つだけ読み込むことを推奨しています。すべてのパラメータが正しくレンダリングされる場合に限り、iFrameを使用してMasterTagを読み込んでください。
</blockquote>


## ロードルール

タグをすべてのページにロードするか、タグがロードされる条件を構成します。ロードルールについての詳細は、[ロードルール](https://docs.tealium.com/about-load-rules/)のドキュメンテーションを参照してください。

**ベストプラクティス**：

* MasterTagの場合、デフォルトの「すべてのページにロード」ルールを選択します。
* PPSタグタイプの場合、注文確認ページ（「ありがとう」または「領収書」ページなど）でタグをロードするロードルールを構成します。
* PPLタグタイプの場合、登録ページでタグをロードするロードルールの条件を構成します。

## データマッピング

[マッピング](https://docs.tealium.com/about-data-mappings/)は、データレイヤーのデータソースからベンダータグの対応する宛先変数にデータを送信するシンプルなプロセスです。このタグの宛先変数は、マッピングツールボックスで利用できます。使いやすさのために、変数は標準とMasterTagの2つのカテゴリに分けられています。


<blockquote>
このタグは、[E-Commerce Extension](https://docs.tealium.com/e-commerce-extension/)が事前に構成されていることを必要とします。適切な注文と製品変数（`_corder`、`_ccurrency`、`_cprod`、`_cprice`、`_cquan`）を構成することで、エクステンションが自動的にそれらの値を対応する宛先にマップすることが可能になります。
</blockquote>


### 標準タブ

PPSまたはPPLタグを使用している場合、このタブの宛先変数にマップします。

| パラメータ | 説明 |
|---|---|
| カテゴリID | リードまたはセールが属するカテゴリの識別子。 |
| 顧客ID | 顧客の一意の識別子。 |
| 注文ID | 注文の識別子。 |
| サブオーダーID | 注文に関する追加情報の識別子。 |
| 通貨記号 | 製品に適用される通貨。例：USドルの場合はUSD。 |
| 合計価格 | 注文内の製品の合計価格。 |
| コミッション固定 | 注文に適用される固定コミッション。 |
| コミッションパーセント | 合計価格に基づくコミッションのパーセンテージ。 |
| セッションID | トランザクションが発生したセッションの識別子。 |
| パートナーID | Zanoxによって割り当てられたパートナー識別子の値。 |
| レビューノート | 注文に対するコメントや個別の備考 |
| 満期日 | 注文の確認日。 |

### MasterTagタブ

MasterTagを使用している場合、このタブの宛先変数にマップします。

| パラメータ | 説明 |
|---|---|
| zx_class (div container class id) | divコンテナの識別子。 |
| zx_identifier | 製品の識別子。 |
| zx_fn | 製品名。 |
| zx_brand | 製品のブランド名。 |
| zx_category | 製品が属するカテゴリ。 |
| zx_amount | 数値としての製品価格 例：2.99。 |
| zx_price | 通貨記号とともに表示される製品価格 例：2.99$ |
| zx_currency | 注文に適用される通貨 例：USD |
| zx_description | 製品の説明。 |
| zx_url | 製品のページのURL。 |
| zx_photo | 製品の画像のURL。 |
| zx_search_query | 検索語/テキスト。 |
