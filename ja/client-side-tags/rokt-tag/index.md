RoktのSDKとAPIのスイートについての詳細は、[Rokt統合ガイド](https://docs.rokt.com/docs/developers/integration-guides/overview%20)をご覧ください。

## タグのヒント

* マッピングを使用して、標準の構成値を上書きし、イベントをトリガーします。

## タグの構成

まず、タグマーケットプレイスに移動し、Roktタグを追加します。詳細については、[タグの管理](https://docs.tealium.com/manage-tags/))を参照してください。

タグを追加したら、以下の構成を行います：

* **RoktアカウントID**：あなたのRoktアカウントID。
* **2ステップメール**：肯定的なエンゲージメントのみでメールアドレスを送信します。
* **サンドボックスの使用**：テスト用にサンドボックスアカウントにイベントを送信します。

## ロードルール

[ロードルール](https://docs.tealium.com/about-load-rules/)は、サイト上でRoktタグのインスタンスをいつ、どこでロードするかを決定します。

コンバージョンページでRoktタグをロードするロードルールを作成します。

## データマッピング

マッピングは、[データレイヤー変数](https://docs.tealium.com/data-layer-variables/)からベンダータグの対応する宛先変数にデータを送信するプロセスです。変数をタグの宛先にマッピングする方法については、[データマッピング](https://docs.tealium.com/ja/iq-tag-management/data-mappings/manage/)を参照してください。

利用可能なカテゴリは以下の通りです：

### 標準

| 変数           | タイプ | 説明   |
|:-------------------|:--------------|:--------------|
| `email`| 文字列 |メールアドレス                   |
| `firstname`| 文字列 |名前          |
| `lastname`| 文字列 |姓            |
| `mobile`| 文字列 |携帯電話番号         |
| `zipcode`| 文字列 |ZIPまたは郵便番号    |
| `country`| 文字列 | ISO Alpha-2国コード    |
| `language`| 文字列 |言語             |
| `sandbox`   |ブール値 |テスト用にイベントをサンドボックスアカウントに送信したい場合は`True`を選択します。        | 

### E-コマース

| 変数                        | 属性           | タイプ | 説明                                             |
|:--------------------------------|:--------------------|:--------------------|:--------------------------------------------------------|
| 確認番号             | `confirmationref` | 数値 | この変数は`_corder`変数を上書きします。確認参照番号。      |
| 金額                          | `amount`          | 数値 |この変数は`_ctotal`変数を上書きします。取引金額。       |
| 通貨                          | `currency`        | 文字列 |この変数は`_ccurrency`変数を上書きします。3文字の通貨コード（例：`USD`）。     |
| 支払いタイプ                    | `paymenttype`    | 文字列| 支払いタイプ（例：`クレジットカード`）。 |                                                         |
| ccbin                           | `ccbin`           | 文字列 | クレジットカードBIN。                                                         |
| 商品価格のリスト             | `price`          | 数値の配列 | この配列は`_cprice`変数を上書きします。各商品の価格のリスト。   |
| 商品数量のリスト          | `quantity`        | 数値の配列 |この配列は`_cquan`変数を上書きします。各商品の価格のリスト。      |
| 商品主要カテゴリのリスト   | `majorcat`        | 文字列の配列 |この配列は`_ccat`変数を上書きします。各商品のカテゴリのリスト。       |
| 商品主要カテゴリIDのリスト | `majorcatid`      | 文字列の配列 | 各商品の主要カテゴリIDのリスト。 |
| 商品副カテゴリのリスト   | `minorcat`        | 文字列の配列 | この配列は`_ccat2`変数を上書きします。各商品の副カテゴリのリスト。      |
| 商品副カテゴリIDのリスト | `minorcatid`      | 文字列の配列 | 各商品の副カテゴリIDのリスト。                                                |
| 商品名のリスト           | `productname`     | 文字列の配列 | この配列は`_cprodname`変数を上書きします。各商品の名前のリスト。|
| 商品SKUのリスト               | `sku`             | 文字列の配列 | この配列は`_csku`変数を上書きします。各商品のSKUのリスト。     |
