---これは、[現在のTealium iQリビジョンAPI](https://docs.tealium.com/about-iq-revisions/)の古いバージョンです。

_リビジョン_は、iQプロファイルの保存されたバージョンです。"名前を付けて保存..."が実行されるたびに新しいリビジョンが作成されます。そして、各バージョンが更新されたり公開されたりすると、その状態がリビジョンに反映されます。リビジョンの状態には、公開された環境、ユーザーコメント、タイムスタンプ、生成されたJavaScriptファイルのチェックサムなど、バージョンに関する情報が含まれています。

_リビジョンバンドル_は、リビジョンに関連する公開ファイルを含むzipファイルです。


<blockquote>
**開始する前に:** iQリビジョンAPIは、データを返す前に有効化する必要があります。このエンドポイントを有効化するには、アカウントマネージャーに連絡してください。
</blockquote>
