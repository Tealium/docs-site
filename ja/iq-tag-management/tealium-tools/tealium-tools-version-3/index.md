## 新機能

これは、Google Chrome Web StoreおよびMicrosoft Edgeアドオンで利用可能なTealium Toolsブラウザ拡張のアップデートです。

Tealium Tools バージョン3をインストールするには、を参照してください。

* **ツールカタログ**:  
**ツールカタログ**はカスタムツールを置き換え、このタブの下にコミュニティツールがグループ化されています。

* **Manifest V3のサポート**:  
Google Chromeは、すべてのChrome拡張機能に影響を与える重要なアップデートを展開しています。このアップデートは、Tealiumのカスタムツールリポジトリサーバーなどの外部ソースからカスタムJavaScriptをロードする拡張機能の能力を削除します。この強制アップデートは、拡張機能自体に含まれていない潜在的に悪意のあるコンテンツのロードを防ぐことで、Chromeユーザーに追加のセキュリティを提供することを目的としています。詳細については、[Chrome for Developers: Extensions &gt; Manifest V3](https://developer.chrome.com/docs/extensions/develop/migrate/what-is-mv3)を参照してください。

* **サイドバーサポート**:  
ツールウィンドウをChromeのサイドバーで開くことができるようになり、ページコンテンツと並行して作業ができるようになりました。これは、**Web Companion**のようなツールにとって特に便利で、ページとの対話を妨げることなく実行できます。

* **カスタムタイルの並び替え**:  
両方のタブでタイルの順序を並び替えて保持することができます。アクセシビリティのために、iOSに似た「クリックアンドホールドでアンロック」機能を実装しました。タイルを左マウスボタンで3秒間押し続けると「ウォブルモード」が有効になり、タイルを並び替えることができます。これにより、誤って並び替えることを防ぎつつ、使いやすさが向上します。

* **SSOログインサポート**:  
シングルサインオン(SSO)機能がより信頼性の高いものになりました。SSOドメインを構成し、シームレスにログインできます。ただし、アカウントでSSOが有効になっていることを確認してください。パスワードおよびMFAログインも引き続きサポートされています。SSOの有効化については、カスタマーサクセスマネージャーにお問い合わせください。

* **uTagデバッガー**:  
**ホーム**タブに公式ツールとして**uTagデバッガー**を含めることができるようになりました。これは、デプロイメントのトラブルシューティングと検証に役立つ貴重なリソースです。

* **組み込みフィードバックフォーム**:  
ユーザーアバターをクリックしてアクセスする管理セクションには、**About**と**Feedback**の2つの新しいタブがあります。フィードバックタブは現在、ツール情報を事前に入力して効果的にトラブルシューティングを支援するバグ提出をサポートしています。**About**タブは、サポートチケットおよびコミュニティツールに関する情報とビルドの詳細を提供します。

* **廃止された機能**:  
以下の機能が削除されました：
  * **マルチパブリッシュ**
  * **CDHコピー＆ペースト**
  * **スキャンコンパニオン**
  * **Verify - テストレコーダー**
  この変更によりワークフローに影響が出た場合は、サポートにお問い合わせください。

* **更新されたツール**:  
  * **Tealium University**、**Tealium Docs**、および**UTag Debugger**は、別々の起動タイルとして利用可能になりました。
  * **TLC**タイルは**ナレッジベース**に名前が変更されました。

* **アーリーアクセスツール**:  
アーリーアクセスツールには、公開されているがリリースの初期段階にあることを示すコールアウトバブルが含まれるようになりました。
