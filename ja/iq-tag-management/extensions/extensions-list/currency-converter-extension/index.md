## 要件

* utag v4.38以降。`utag.js`テンプレートの更新についての詳細は、ナレッジベースの記事[utag.jsの最新バージョンへの更新のベストプラクティス](https://support.tealiumiq.com/en/support/solutions/articles/36000363470)を参照してください。
* [Tealium通貨変換タグ](https://docs.tealium.com/tealium-currency-converter-tag/)（すべてのタグまたは特定のタグにスコープを構成できます）

## 動作原理

通貨変換エクステンションはTealium通貨変換タグと連携して動作し、エクステンションに為替レートを提供します。このエクステンションは、ある通貨のデータレイヤー変数の値を別の通貨に変換します。エクステンションは多くのデータレイヤー変数の変換を構成することができ、各変換はロードルールによって制御され、必要な時にのみ変換を実行します。

## エクステンションの使用方法

始める前に、[エクステンションの動作方法]()について熟知してください。

エクステンションを追加したら、**通貨変換を追加**をクリックして新しい変換を追加し、次のフィールドを構成します（必要に応じてこのステップを繰り返します）：

* **変換する**: 変換する値を含むデータレイヤー変数。
* **条件**: 変換をトリガーするロードルール。
* **変換前の通貨**: 現在の値の通貨。
* **変換後の通貨**: 新しい、変換された値の通貨。

### 例

あなたのサイトでは、顧客は任意の通貨で購入が可能ですが、あるタグベンダーは常にUSD値として金額を送信することを期待しています。以下は、多くの通貨からUSDにデータレイヤー変数を変換するために必要なコンポーネントです：

* **通貨ロードルール**: 顧客が使用する可能性のある各通貨に対してロードルールを作成します。例えば：

  
  [
    [
      {
        "input": "site_currency",
        "operator": "equals (ignore case)",
        "filter": "GBP"
      }
    ]
  ]
  
    
* **通貨変換エクステンション**: サイトがサポートする各通貨および変換が必要な各データレイヤー変数に対して通貨変換を追加します。この例では、すべての値がUSDで送信されることが求められる特定のタグにスコープが構成されていることに注意してください。
