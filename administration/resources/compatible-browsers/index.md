## Supported browsers

We support the following browsers:

* Google Chrome: Windows and Mac desktop versions (Recommended)
* Firefox: Windows and Mac desktop versions
* Microsoft Edge: Desktop version

The five most recent versions of the above browsers are supported.

Because Chrome and Firefox perform upgrades to ensure that all users are current, explicit testing is only performed on current versions.

The interface is designed for a minimum browser width of 1440 pixels.

### Deprecated browsers

* Support for Safari will end on January 1, 2024.
* Support for Internet Explorer 11 ended on November 1, 2021.

## Terms of support

Tealium browser support is as follows:

* 60 days notice will be provided if support for a browser version is to be discontinued.
* Tealium reserves the right to release certain non-essential features and applications on more specific browsers or browser versions.
* Support for two earlier versions or older only covers essential functionality of the product, not necessarily proper rendering of the intended look and feel.
* Tablet or mobile device browsers are not directly supported.

This browser support does not apply to the Universal Tag (`utag.js`), which has widespread browser support.
