## Overview

The navigation menu is now organized by actions instead of product names. Categories are grouped by common jobs across different user roles.

![](/images/administration/navigation-menu-after.png) 

* **Tag Management**  
Manage your client-side tags, rules, events, and consent.
* **Connect**  
Manage inbound and outbound data integrations, including data collection and connectors.
* **Govern**  
Manage data quality, privacy, and consent, including event specifications, consent orchestration, and visitor search.
* **Transform**  
Manage data attributes and create data assets with enrichments such as rules, functions, and predictive scoring.
* **Activate**  
Manage data activation, including audience creation, audience jobs, and visitor profile activation through the Moments API.
* **Validate**  
Test and verify use cases in production using tools such as trace, live events, and live visitors (formerly called “visitor profile sampler”).
* **Store**  
Manage database storage products in one place. Features previously part of DataAccess are now included here.
* **Analyze**  
Use reporting and analytics tools to monitor performance, generate insights, and measure results across data and audiences.

### Availability

The new navigation will be available starting October 2, 2025. To ensure a smooth rollout, we are enabling it gradually. You will be notified several weeks before the new navigation is enabled for your account.

### Scope

When you enable the new navigation, the change applies only to your user account. It affects all accounts that you access, but does not affect other users.

![](/images/administration/new-navigation-toggle.png)

### Provide feedback

You can share feedback directly from the new navigation toggle:

1. Open the **User Admin** menu and hover over the **New navigation** toggle.
1. Select **Share your feedback** and enter your feedback.
1. Select **Send feedback**.
