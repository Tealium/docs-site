## Features

* **[Explicit Consent Prompt Manager]()**  
Learn how to deploy a prompt to your website that allows users to opt-in or opt-out of tracking. This tool offers full access to customize the code behind the scenes and options for translated content.
* **[Consent Preferences Manager]()**  
Learn how to deploy a tracking preferences option to your website that will allow customers to specify which categories of tags they allow. This tools offers full access to customize the code that generates the preferences pop-up and the ability to translate content for your global customers. 
* **[Opt-out Privacy Banner and Popup]()**
Learn how to deploy a privacy banner and popup to your website to allow users to opt-in or opt-out of the sale of personal information. This tool offers full access to customize the code behind the scenes and options for translated content.
* **[Consent Register](https://docs.tealium.com/consent-register/)**
Sharing consent decisions across your website allows functionalities like Google consent mode to access these decisions for appropriate actions. The consent register acts as a standardization layer for consent signals, and makes those standardized consent signals easily available on your webpage. 
<blockquote>
Consent register requires the consent management template version 3.1.0 or later. To update this template, see [Update a template](https://docs.tealium.com/manage-templates/#update-a-template).
</blockquote>
