## Configure Settings

Go to the Connector Marketplace and add a new connector. Read the [Connector Overview]() article for general instructions on how to add a connector.

After adding the connector, configure the following settings:

* **Branch Key**  
This value is found on the **Profile** page of the **Branch Account Manager**.

## Action Settings - Parameters and Options

Click **Next** or go to the **Actions** tab. This is where you configure connector actions.

This section describes how to set up parameters and options for each action.

&lt;/li&gt;&lt;li&gt;**Locally Indexable** &lt;ul&gt;&lt;li&gt;If value is `true`, content can be indexed for local (device) use.&lt;/li&gt;&lt;li&gt;If value is `false`, cannot index for local use.&lt;/li&gt;&lt;/ul&gt; &lt;/li&gt;&lt;li&gt;**Price** &lt;ul&gt;&lt;li&gt;Price for the product/content.&lt;/li&gt;&lt;/ul&gt; &lt;/li&gt;&lt;li&gt;**Quantity** &lt;ul&gt;&lt;li&gt;Quantity of the item to be ordered.&lt;/li&gt;&lt;li&gt;Example: for `PURCHASE`, `ADD_TO_CART`&lt;/li&gt;&lt;/ul&gt; &lt;/li&gt;&lt;li&gt;**SKU** &lt;ul&gt;&lt;li&gt;Product SKU or product ID.&lt;/li&gt;&lt;/ul&gt; &lt;/li&gt;&lt;li&gt;**Product Name** &lt;ul&gt;&lt;li&gt;Name of the product.&lt;/li&gt;&lt;/ul&gt; &lt;/li&gt;&lt;li&gt;**Product Brand** &lt;ul&gt;&lt;li&gt;Brand of the product.&lt;/li&gt;&lt;/ul&gt; &lt;/li&gt;&lt;li&gt;**Product Category** &lt;ul&gt;&lt;li&gt;Category of a product. if applicable.&lt;/li&gt;&lt;li&gt;One of:  &lt;ul&gt;&lt;li&gt;`ANIMALS_AND_PET_SUPPLIES`&lt;/li&gt;&lt;li&gt;`APPAREL_AND_ACCESSORIES`&lt;/li&gt;&lt;li&gt;`ARTS_AND_ENTERTAINMENT`&lt;/li&gt;&lt;li&gt;`BABY_AND_TODDLER`&lt;/li&gt;&lt;li&gt;`BUSINESS_AND_INDUSTRIAL`&lt;/li&gt;&lt;li&gt;`CAMERAS_AND_OPTICS`&lt;/li&gt;&lt;li&gt;`ELECTRONICS`&lt;/li&gt;&lt;li&gt;`FOOD_BEVERAGES_AND_TOBACCO`&lt;/li&gt;&lt;li&gt;`FURNITURE`&lt;/li&gt;&lt;li&gt;`HARDWARE`&lt;/li&gt;&lt;li&gt;`HEALTH_AND_BEAUTY`&lt;/li&gt;&lt;li&gt;`HOME_AND_GARDEN`&lt;/li&gt;&lt;li&gt;`LUGGAGE_AND_BAGS`&lt;/li&gt;&lt;li&gt;`MATURE`&lt;/li&gt;&lt;li&gt;`MEDIA`&lt;/li&gt;&lt;li&gt;`OFFICE_SUPPLIES`&lt;/li&gt;&lt;li&gt;`RELIGIOUS_AND_CEREMONIAL`&lt;/li&gt;&lt;li&gt;`SOFTWARE`&lt;/li&gt;&lt;li&gt;`SPORTING_GOODS`&lt;/li&gt;&lt;li&gt;`TOYS_AND_GAMES`&lt;/li&gt;&lt;li&gt;`VEHICLES_AND_PARTS`&lt;/li&gt;&lt;/ul&gt; &lt;/li&gt;&lt;/ul&gt; &lt;/li&gt;&lt;li&gt;**Product Variant** &lt;ul&gt;&lt;li&gt;Variant of the product.&lt;/li&gt;&lt;li&gt;Example: `red`&lt;/li&gt;&lt;/ul&gt; &lt;/li&gt;&lt;li&gt;**Rating Average** &lt;ul&gt;&lt;li&gt;Average rating of the item.&lt;/li&gt;&lt;/ul&gt; &lt;/li&gt;&lt;li&gt;**Rating Count** &lt;ul&gt;&lt;li&gt;Number of ratings for the item.&lt;/li&gt;&lt;/ul&gt; &lt;/li&gt;&lt;li&gt;**Rating Max** &lt;ul&gt;&lt;li&gt;Maximum possible rating for the item.&lt;/li&gt;&lt;li&gt;Example: `5.0` if 5 stars is highest possible rating.&lt;/li&gt;&lt;/ul&gt; &lt;/li&gt;&lt;li&gt;**Creation Timestamp** &lt;ul&gt;&lt;li&gt;Time the content was created.&lt;/li&gt;&lt;/ul&gt; &lt;/li&gt;&lt;li&gt;**Exp Date** &lt;ul&gt;&lt;li&gt;The last time after which this content is no longer valid.&lt;/li&gt;&lt;li&gt;`Null` /  means no limit.&lt;/li&gt;&lt;li&gt;Should rarely be set.&lt;/li&gt;&lt;/ul&gt; &lt;/li&gt;&lt;li&gt;**Condition** &lt;ul&gt;&lt;li&gt;For auctions, whether the item is new, good, acceptable, etc.&lt;/li&gt;&lt;li&gt;One of:  &lt;ul&gt;&lt;li&gt;`OTHER`&lt;/li&gt;&lt;li&gt;`NEW`&lt;/li&gt;&lt;li&gt;`EXCELLENT`&lt;/li&gt;&lt;li&gt;`GOOD`&lt;/li&gt;&lt;li&gt;`FAIR`&lt;/li&gt;&lt;li&gt;`POOR`&lt;/li&gt;&lt;li&gt;`USED`&lt;/li&gt;&lt;li&gt;`REFURBISHED`&lt;/li&gt;&lt;/ul&gt; &lt;/li&gt;&lt;li&gt;**Latitude** &lt;ul&gt;&lt;li&gt;Latitude for a restaurant, business, room (hotel), etc.&lt;/li&gt;&lt;/ul&gt; &lt;/li&gt;&lt;li&gt;**Longitude** &lt;ul&gt;&lt;li&gt;Longitude for a restaurant, business, room (hotel), etc.&lt;/li&gt;&lt;/ul&gt;
