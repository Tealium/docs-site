Twitter Campaigns Connector Setup Guide

This article describes how to set up the Twitter Campaigns connector.

Connector Actions

Action Name AudienceStream EventStream
Add Targeting Criteria to Line Item ✓ ✗
Remove Targeting Criteria from Line Item ✓ ✗

Configure Settings

Go to the Connector Marketplace and add a new connector. Read the Connector Overview article for general instructions on how to add a connector.

After adding the connector, configure the following settings:

  • Ad Account ID
    • Required.
    • Provide your Twitter Ad Account ID.
    • Your Account ID can be located in your the URL for your account when logged in.
    • Example: If the URL is https://ads.twitter.com/accounts/1234abcd/, the account ID is 1234abcd.

Action Settings - Parameters and Options

Click Next or go to the Actions tab. This is where you configure connector actions.
