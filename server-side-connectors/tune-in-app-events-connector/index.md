## Connector Actions

| **Action Name**       | **AudienceStream** | **EventStream** |
|:----------------------|:-------------------|:----------------|
| Send Predefined Event | ✓                  | ✓               |
| Send Custom Event     | ✓                  | ✓               |

## Requirements

* TUNE Account
* Email and Password
* Private/API key
* Public/Consumer key

To obtain the listed keys, you must send a request to TUNE Support from the email address that is linked to the TUNE account, and they will create the keys for you.

## Configure Settings

Go to the Connector Marketplace and add a new connector. Read the [Connector Overview]() article for general instructions on how to add a connector.

After adding the connector, configure the following settings:

* **Private API key**
  * Required
  * Provide your private API key.
  * For additional information, see: [Obtaining your private API key](https://tune.docs.branch.io/measurement-docs/obtaining-your-private-api-key/)

* **Public Consumer key**
  * Required
  * Provide your public consumer key.

## Action Settings - Parameters and Options

Click **Next** or go to the **Actions** tab. This is where you configure connector actions.

This section describes how to set up parameters and options for each action.
