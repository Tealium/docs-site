## API information

This connector uses the following vendor API:

* API Name: Facebook Graph API - Marketing API
* API Version: v25.0
* API Endpoint: `https://graph.facebook.com`

## How it works

Report app installs and in-app conversions through this API and send data back to Facebook Analytics. Use this on mobile devices without the Facebook SDK. For detailed information, see the [Facebook App Events API](https://developers.facebook.com/docs/marketing-api/app-event-api#install_parameters) documentation.

## Configuration

Go to the Connector Marketplace and add a new connector. Read the [Connector Overview]() article for general instructions on how to add a connector.

After adding the connector, configure the following settings:

* **Application ID**: The App ID listed on your app dashboard at Facebook for Developers.

## Actions

| Action Name| AudienceStream | EventStream|

Click **Next** or go to the **Actions** tab. This is where you configure connector actions.
