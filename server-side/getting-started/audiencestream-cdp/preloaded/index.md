AudienceStream and EventStream contain a set of preloaded visit and visitor attributes that many of our customers need. Each attribute is preconfigured with statistical information that can help you determine useful information about each visitor or visit.

![](https://docs.tealium.com/images/server-side/getting-started/audiencestream-cdp/preloaded-attributes.png)

You can use these attributes as a starting point for your own attributes. Because preloaded attributes cannot be changed, duplicate the preloaded attribute and then customize the copy to fit your needs.

For more information about these attributes, see [Preloaded attributes](https://docs.tealium.com/preloaded-attributes/).
