---
title: Enable Data Layer Enrichment
description: This article describes how to enable Data Layer Enrichment.
url: https://docs.tealium.com/server-side/attributes/data-layer-enrichment/enable/
---
To enable data layer enrichment, you must select an AudienceStream profile from which to import the attributes. By default, Tealium iQ loads attributes from the main AudienceStream profile, but an alternative profile can be set in the Publish Configuration area of Tealium iQ.

To set the AudienceStream profile used for data layer enrichment:

1. In the admin menu, click **Configure Publish Settings**. The **Publish Configuration** dialog appears.
1. On the **General Publishing** tab, scroll to the bottom of the **Implementation** section.
1. Select the appropriate **Data Layer Enrichment Profile**.
1. Click **Apply** and continue to save and publish your changes to the **Prod** environment.
