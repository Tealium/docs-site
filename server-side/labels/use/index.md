## Filter view by label

1. Click one of the following:
    * **Live Events > Event Feed**
    * **Event Attributes**
    * **Visitor/Visit Attributes**
1. Under the **Filter** sidebar, navigate to the **Labels** section.
1. Select a label from the list to filter by that label.
1. To remove the filter, uncheck the label.

## Search by label

Use the following steps to search based on labels:

1. In the search bar at the top of the screen, enter the label you want to search by.  
    ![](https://docs.tealium.com/images/iq-tag-management/whiteui-tiq-lables-searchtiq.png)  
The search bar displays results for labels applied to data sources, event feeds, event specs, and attributes.
1. Select any label from the search results to go to that area of the interface, filtered by the label searched for.
