## Credentials

You will need the following credentials to configure the file service in the File Import configuration:

* Host name
* User name
* Authentication method  
**Password**, **Upload Private Key File**, and **Generate Key Pair** are supported authentication methods.

Additionally, you can specify the port and add a subdirectory.
