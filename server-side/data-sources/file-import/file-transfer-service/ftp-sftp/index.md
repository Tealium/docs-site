## SFTP

### Credentials

You will need the following FTP credentials to configure the file service in the File Import configuration:

* Host name
* User name
* Authentication method  
**Password**, **Upload Private Key File**, and **Generate Key Pair** are supported authentication methods.

Additionally, you can specify the port and add a subdirectory.

## FTP

By using an FTP connection to connect to a file import data source, all data, including FTP credentials, will be transmitted unencrypted over the internet.

### Credentials

You will need the following FTP credentials to configure the file service in the File Import configuration:

* Host name
* User name
* Password
* Port

Additionally, you can add a subdirectory.
