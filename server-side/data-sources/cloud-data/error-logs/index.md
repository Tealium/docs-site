## View logs

To view error logs, use the following steps:

1. From the **Data Sources** dashboard, expand the cloud data source you want to view.
1. Under **Errors**, click **Error Logs** or click the edit icon and select the **Error Logs** tab.
1. Select the error log you want to view and preview or export the log.

## Export logs

You can export logs for the following time ranges:

* Last 30 minutes
* Last hour
* Last 4 hours
* Custom

To reduce file sizes, exports are limited to a time range of four hours or shorter.

Complete the following steps to export logs:

1. From the **Error Logs** tab, click **Export Logs**.
1. Select a time range or select **Custom** to choose the start and end date and time. 
1. Select the error types to include in the log.
1. Click **Request Export**.

There is a short delay while the logs are processed. 
You can view your exported logs from the past six days by navigating to **Error Logs&gt;View Exported Logs**. Logs are stored for 30 days.
