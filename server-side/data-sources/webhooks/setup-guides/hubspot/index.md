## Prerequisites

* HubSpot Marketing Professional, Sales Professional, or Enterprise account
* Tealium EventStream and/or Tealium AudienceStream account

## How it works

HubSpot offers a webhook API that sends outgoing requests to an endpoint that you specify. These requests act like push notifications to EventStream to inform it about things that happened in your HubSpot account. When you add the HubSpot data source in the UDH, a unique endpoint will be generated that you will use to configure the HubSpot webhook.

## Add the HubSpot data source

Add the HubSpot data source before proceeding. The HubSpot data source generates a unique URL to use as the HTTP POST URL in your HubSpot configuration. The URL generated is in the following format:

```
https://collect.tealiumiq.com/integration/event/ACCOUNT/PROFILE/DATA_SOURCE_KEY
```

## Configuring HubSpot

Once you have your data source endpoint, go to your HubSpot account to create the webhook.

1. Log in to your [HubSpot account](https://login.hubspot.com/).
    * In your HubSpot Marketing Professional or Enterprise account, navigate to **Contacts &gt; Workflows**.
    * In your HubSpot Sales Professional account, navigate to **Sales Tools &gt; Workflows**.

1. Click the name of the workflow you'd like to edit.
1. Click the plus (**+** ) icon to add an action and in the right pane;and select **Trigger a webhook**.
1. From the drop-down list, choose **POST**.
1. Enter the following for the WebHook URL:  
      ```
      https://collect.tealiumiq.com/integration/event/{account}/{profile}/{unique_id}
      ```

1. Click **Save**.
