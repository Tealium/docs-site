You can edit your data source settings to add notes, labels, a custom logo URL, or a custom URL to link to the data source platform.

Use the following steps to edit the data source settings: 

1. Click the name of the data source to expand the details for that data source.
1. Click the pencil icon in the toolbar that displays on the bottom right of the data source.  
The **Data Source Settings** dialog displays. 
      You can optionally hover over the logo to display a pencil icon over the logo and then click. 
      ![](/images/server-side/whiteui-editdatasource.png)  
1. In the **Notes** field, enter helpful note related to the data source.
1. To edit the labels, click **Add Labels**.
1. In the **Custom Logo URL** field, enter a valid URL to display your custom logo.  
      The logo file name must end with .jpg, .png, or .gif.
      The logo URL must start with `https://`
1. In the **Custom Homepage URL** field, enter a valid URL for your homepage.
1. Click **Save**.
1. Click **Finish**.
