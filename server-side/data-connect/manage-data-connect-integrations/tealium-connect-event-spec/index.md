After you create a Tealium Connect data source, create a custom event specification for each type of event you track from Data Connect. Use event specifications to validate incoming events in [Live Events](https://docs.tealium.com/about-live-events/).

For more information, see [Event Specifications](https://docs.tealium.com/about-event-specifications/).

Complete the following steps to add a custom event specification:

1. Navigate to **Validate > Event Specs**.
1. Click **+ Add Event Specification** and select **Add Custom Specification** from the drop down.
1. Enter a name for the event.
1. Select the attributes from the **Attributes** drop down lists that you want included in your specification.
1. Click **Finish**.
1. Navigate to the **Data Sources** dashboard and select the appropriate data source for your event specification. 
1. In the expanded data source view, click **Add/Remove Event Specs**.
1. In the **Add From Marketplace** screen under **Categories**, select the **Custom** category.
1. Select the custom event specification you just created.
1. Click **Save**.
