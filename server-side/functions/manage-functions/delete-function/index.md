---
title: Delete a function
description: This article provides instructions for deleting a function from the **Functions Overview** page and from the functions code editor.
url: https://docs.tealium.com/server-side/functions/manage-functions/delete-function/
---
 To delete a function from the list of functions, follow these steps:

1. Click the menu for the function and then click **Delete**.
1. In the **Delete Function** dialog, click **Delete**.
1. Click **Save/Publish**.

To delete a function from the code editor, follow these steps:

1. Click **Delete** in the function title bar.
1. In the **Delete Function** dialog, click **Delete**.
1. Click **Save/Publish**.
