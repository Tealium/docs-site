A _revision_ is a saved version of your iQ profile. A new revision is created each time a "Save as..." is performed. Then as each version is updated and/or published, the status is reflected in the revision. The revision status contains information about the version, such as published environments, user comments, timestamps, and checksums for the generated JavaScript files.

A _revision bundle_ is a zip file containing the published files related to the revision.


<blockquote>
**Before You Begin:** The iQ revisions API must be activated before it will return data. Contact your account manager to activate this endpoint.
</blockquote>
